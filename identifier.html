14. Mengapa identifier tidak boleh sama dengan kata kunci (keyword) dalam bahasa C!


Jawaban & Penjelasannya

Identifier dan kata kunci memiliki peran yang berbeda dalam bahasa C. Identifier digunakan sebagai nama untuk variabel, fungsi, dll, sedangkan kata kunci memiliki arti khusus dalam bahasa dan digunakan untuk menyatakan aksi atau pengendalian aliran program. Jika identifier memiliki nama yang sama dengan kata kunci, compiler akan menganggap bahwa kita mencoba mengubah arti dari kata kunci, yang menyebabkan kesalahan sintaks dan kesalahan kompilasi. Oleh karena itu, untuk menghindari kesalahan dalam pemrograman, identifier tidak boleh sama dengan kata kunci.

SEMOGA MEMBANTU

Dikarenakan apabila kamu menggunakan keyword sebagai identifier, maka program kamu akan tidak berjalan sesuai dengan semestinya.

PEMBAHASAN

Keyword adalah kata-kata kunci pada sebuah bahasa pemrograman yang sudah "dipesan" oleh bahasa pemrograman tersebut untuk menjalankan suatu perintah khusus.

Sedangkan Identifier merupakan sebuah nama yang kita berikan terhadap suatu variabel, fungsi, program, dsb.

Ada 32 keyword pada bahasa pemrograman C, yakni:

auto, break, case, char, const, continue, default, do, double, else, enum, extern, float, for, goto, if, int, long, register, return, short, signed, sizeof, static, struct, switch, typedef, union, unsigned, void, volatile, dan while.

Kata-kata terebut tidak dapat digunakan sebagai nama variabel, fungsi, program dsb. dikarenakan compiler (pengubah kode pemrograman menjadi kode mesin) akan bingung dan mengira variabel/fungsi tersebut merupakan salah satu perintah yang khusus tersebut.

Salah satu cara agar kamu dapat menggunakan keyword-keyword  tersebut pada suatu variabel/fungsi adalah dengan cara menambahkan tanda underscore (_) dan compiler tidak akan lagi bingung dan memastikan bahwa nama tersebut merupakan sebuah variabel/fungsi dan bukan salah satu keyword khusus. Sebagai contoh: _short, _float, dsb.

15. Kenapa huruf besar atau kecil dianggap sebagai dua identifier yg berbeda?


Dalam banyak sistem komputer, besar kecilnya huruf dianggap sebagai dua identifier yang berbeda karena sistem komputer membedakan antara huruf besar dan kecil sebagai bagian dari representasi karakter. Ini membuat identitas unik bagi setiap string yang berbeda dan memungkinkan sistem untuk membedakan antara nama file, variabel, dan lainnya yang memiliki huruf besar dan kecil yang berbeda.

17. Jelaskan konsep identifier global dan lokal pada prosedur


Identifier global adalah identifier yang dikenali di semua bagian program, contohnya adalah nilaiAkhir dan nilaiHuruf

Identifier lokal adalah identifier yang hanya dikenali oleh sub program yang mendeklarasikannya, contohnya adalah uts, uas dan tugas yang hanya dikenali oleh prosedur hitungNilai.

Pembahasan

Pemrograman Java adalah salah satu jenis bahasa pemrograman yang dapat dijalankan di berbagai komputer termasuk telepon genggam.  

Prosedur

Prosedur adalah sebutan untuk fungsi yang tidak mengembalikan nilai. Fungsi ini biasanya ditandai dengan kata kunci void.

Penulisan prosedur pada pemrograman Java

Prosedur Hitung Nilai

Procedure HITUNGNILAI

Input(uts, uas, tugas)

nilaiAkhir_0.3uts+o.5uas+0.2tugas

Output(nilaiAkhir)

Prosedur Nilai Huruf

Procedure NILAIHURUF

If nilaiAkhir>=80 then nilaiHuruf_’A’

Else if If nilaiAkhir>=70 then nilaiHuruf_’B’

Else if If nilaiAkhir>=50 then nilaiHuruf_’C’

Else if If nilaiAkhir>=30 then nilaiHuruf_’D’

Else nilaiHuruf_’E’

End If

Output(nilaiHuruf)

     import java.util.Scanner;

     public class ContohProsedur{

                 //deklarasi variabel global yang bisa dikenali disemua bagian program

                 public static double nilaiAkhir=0;

                 public static char nilaiHuruf=' ';

                 //prosedur Hitung Nilai

                 public static void HitungNilai(){

                             Scanner keyboard=new Scanner(System.in);

                             System.out.print("Nilai UTS : ");

                         double uts = keyboard.nextDouble();

                         System.out.print("Nilai UAS : ");

                         double uas = keyboard.nextDouble();

                         System.out.print("Nilai Tugas : ");

                         double tugas = keyboard.nextDouble();

                         nilaiAkhir=0.3*uts+0.5*uas+0.2*tugas;

                         System.out.printf("Nilai Akhir %.2f : \n" ,nilaiAkhir);

             }

             //Prosedur Nilai Huruf

             public static void NilaiHuruf(){

                         if (nilaiAkhir>=85) nilaiHuruf='A';

                         else if (nilaiAkhir>=70) nilaiHuruf='B';

                         else if (nilaiAkhir>=50) nilaiHuruf='C';

                         else if (nilaiAkhir>=30) nilaiHuruf='D';

                         else nilaiHuruf='E';

                         System.out.println("Nilai Huruf : "+nilaiHuruf);

             }

             //Program Utama

             public static void main(String[] args){

             HitungNilai();

                         NilaiHuruf();

             }

}

Fungsi

Fungsi adalah sub program yang mengembalikan (return) sebuah nilai bertipe data primitif (int, float, double, boolean, string, atau char)

import java.util.Scanner;

public class ContohFungsi{

           //Fungsi Hitung Nilai

           public static double HitungNilai(double a, double b,double c){

                       double na = a*0.3+b*0.5+c*0.2;

                       return (na);

           }

           //FungsiNilai Huruf

           public static char NilaiHuruf(double na){

                       char nh = ' ';

                       if (na>=85) nh='A';

                       else if (na>=70) nh='B';

                       else if (na>=50) nh='C';

                       else if (na>=30) nh='D';

                       else nh='E';

                       return(nh);

           }

           //Program Utama

           public static void main(String[] args){

                       Scanner keyboard = new Scanner(System.in);

                       System.out.print("Nilai UTS : ");

                       double uts = keyboard.nextDouble();

                       System.out.print("Nilai UAS : ");

                       double uas = keyboard.nextDouble();

                       System.out.print("Nilai Tugas : ");

                       double tugas = keyboard.nextDouble();

                       double nilaiAkhir = HitungNilai(uts,uas,tugas);

                       System.out.printf("Nilai Akhir : %.2f \n",nilaiAkhir);

                       System.out.println("Nilai Huruf : "+NilaiHuruf(nilaiAkhir));

           }

}

Identifier yaitu merupakan bagian dari statement yang merupakan bagian dari identitas dari sesuatu. Identitas itu sendiri yaitu meliputi:

Nama programNama fungsiNama variabelNama konstanta

Tata cara aturan dari penulisan identifier adalah sebagai berikut:

Karakter yang pertama harus berupa huruf.Karakter yang kedua dan seterusnya bisa saja berupa huruf, angka, atau karakter underscore.Maksimal panjang dari identifier tergantung kepada compiler yang dipakai. Beberapa untuk mendukung 32 karakter, namun kebanyakan bisa saja mendukung sampai dengan 63 karakter. Apabila kita akan membuat identifier dengan panjang yaitu melebihi 63 karakter, maka hanya 63 karakter yang pertama saja yang akan bisa digunakan.Penulisan dari identifier ini tidak boleh menggunakan karakter yang selain angka, huruf dan underscore.
