11. carilah suku ke n-1 dari 10,19,31,46,64


10,19,31,46,64
a=10 (kita simpan dulu)
misal
U1 = 19
U2 = 31
U3 = 46
dan seterusnya

b1=19-10 = 9
b2=31-19 = 12
b3=46-31 = 15
dan seterusnya (membentuk barisan aritmatika)
jika c = beda
maka
c = b2-b1 = b3-b2 = b4-b3 = 3

maka
bn = b1+(n-1)c
bn=9+(n-1)3
bn= 9+3n-3
bn= 3n+6 = 3(n+2)

Sbn = (n/2)[2b1+(n-1)c]
[tex] = \frac{n}{2} (2.9 + (n - 1)3) \\ = \frac{n}{2} (3n + 15)[/tex]

maka
Un = a + Sbn-1
[tex]Un = a + \frac{n - 1}{2} (3(n - 1) + 15)[/tex]
[tex]Un = a + \frac{n - 1}{2} (3n + 12)[/tex]

maka
Un-1 = a + Sbn-1-1 = a + Sbn-2
[tex]Un-1 = a + \frac{n - 2}{2} (3(n - 2) + 15)[/tex]
[tex]Un-1 = a + \frac{n - 2}{2} (3n+ 9)[/tex]

Deret Aritmatika Bertingkat

10, 19, 31, 46, 64, ...
...9....12....15...18....
.......3.....3.....3...

2A = 3 → A = 3/2

3A + B = 9
9/2 + B = 9 → B = 9/2

A + B + C = 10 → C = 4

Un = An² + Bn + C
Un = 3/2 n² + 9/2 n + 4

U(n - 1)
= 3/2 (n - 1)² + 9/2 (n - 1) + 4
= 3/2 n² - 3n + 3/2 + 9/2 n - 9/2 + 8/2
= 3/2 n² - 3/2 n + 1 → jawaban



12. carilah suku ke n-1 dari 10,19,31,46,64


♦ Penyelesaian :

10 = a ← ........ 10, 19, 31, 46, 64
9 = b ← ..............9,..12,..15,..18
3 = c ← .................3,...3,...3

Ditanya : → Un - 1

Pertama, kita cari berapa nilai Un. Un merupakan suku terakhir dari barisan bilangan.

10, .. 19, .. 31, .. 46, .. 64
↓.......↓.......↓......↓.......↓
U1__U2__U3_U4__U5 / Un

Jadi, nilai Un = 64.

= Un - 1
= 64 - 1
= 63 ✔

Nilai beda = U²-U¹ = 14-20 = -6

= Un = a + (n-1).b

= Un = 20 + (n-1).(-6)

b. Rumus jml n suku pertama

= Sn = n/2 (2a + (n-1).b)

= Sn = n/2 (40 + (n-1).(-6))

c. Jumlah 12 suku pertama

= Sn = n/2 (40 + (n-1).(-6))

= S¹² = 12/2 ( 40 + (12 - 1). (-6))

= S¹² = 6 × (-26)

= S¹² = -156
