2. Jika diketahui inflasi yang terjadi pada bulan agustus 2018 20% inflasi pada bulan september 2018 40% dan inflasi pada bulan oktober 2018 50% tentukan besarnya laju inflasi pada bulan september 2018 dan oktober 2018tolong jangan ngasal dan tolong pakai cara​


Jawaban:

Penjelasan:

[tex]Laju \ Inflasi[/tex]

Diketahui:

Inflasi bulan Agustus 2018 sebesar 20%

Inflasi bulan September 2018 sebesar 40%

Inflasi bulan Oktober 2018 sebesar 50%

A. Ditanyakan Laju Inflasi bulan September?

[tex]Laju \ Inflasi = \frac{40 - 20}{20} \ x \ 100 \\\\Laju \ inflasi = \frac{20}{20} \ x \ 100\\\\Laju \ Inflasi = 100[/tex]

Kesimpulan:

Besar laju inflasi pada bulan september adalah sebesar 100%

B. Ditanyakan Laju Inflasi bulan Oktober?

[tex]Laju \ Inflasi = \frac{50 - 40}{40} \ x \ 100 \\\\Laju \ inflasi = \frac{10}{40} \ x \ 100\\\\Laju \ Inflasi = 25[/tex]

Kesimpulan:

Besar laju inflasi pada bulan september adalah sebesar 25%

21. pada bulan maret 2018 indeks harga konsumen sebesar 138 dan indeks harga konsumen bulan yang sama setahun sebelum nya sebesar 131,dari data tersebut laju inflasi pada bulan maret 2018 sebesar​


Jawaban:

pada bulan maret 2018 indeks harga konsumen sebesar 138 dan indeks harga konsumen bulan yang sama setahun sebelum nya sebesar 131,dari data tersebut laju inflasi pada bulan maret 2018 sebesar​ adalah 5,3%

Penjelasan:

Inflasi merupakan suatu keadaan dimana harga barang-barang naik secara umum, untuk menghitung besarnya laju inflasi dapat kita gunakan rumus berikut ini :

Laju inflasi = indeks sekarang - indeks tahun sebelumnya x 100%

                                 indeks harga tahun sebelumnya

                 = 138 - 131 / 131 x 100%

                 = 5,3%

Jadi besarnya lahu inflasi pada bulan maret 2018 adalah sebesar 5,3%
