30. Penduduk dusun A berjumlah 50.000 orang. Setiap tahun penduduk dusun itu bertambah 3.000 orang. Penduduk dusun B berjumlah 160.500 orang dan setiap tahun penduduk dusun B berkurang 3.500 orang. Berapa tahun lagi kedua dusun itu mempunyai jumlah penduduk yang sama?


a1 = 50.000
b1 = 3.000
a2 = 160.500
b2 = -3.500

a1 + (n -  1)b1 = a2 + (n - 1)b2
50.000 + (n - 1)3.000 = 160.500 + (n - 1)(-3.500)
50.000 - 3.000 + 3.000n = 160.500 - 3.500n + 3.500
3.000n + 3.500n =160.500 + 3.500 - 50.000 + 3.000
6.500n = 117.000
n = 117.000/6.500
n = 18

Jadi, jumlah penduduknya sama 18 tahun lagi.
