20. Bahasa Inggrisnya "Pelunasan Pembayaran"repayment ? atau ada istilah yg lebih formal atau sering dipakai​


Pelunasan pembayaran dalam bahasa Inggris disebut repayment

Pembahasan

Anuitas adalah pembayaran atau penerimaan sejumlah uang yang umumnya sama besar dengan periode yang sama untuk setiap pembayarannya.

Jenis - jenis anuitas adalah sebagai berikut:

Anuitas biasa (pembayaran dilakukan setiap akhir periode)Anuitas di muka  (pembayaran dilakukan setiap awal periode)Anuitas di tunda (pembayaran dimuka setelah beberapa periode)

Amortisasi utang untuk anuitas biasa nilai sekarang

Jika hari ini adalah pembayaran uang muka, maka periode berikutnya adalah pembayaran angsuran utama.Pembayaran pertama angsuran sebagian untuk membayar bunga dan sebagian sisanya untuk membayar pelunasan pokok.Sisa saldo utang pada periode tertentu dapat diperoleh dari nilai sekarang dari sisa periode yang belum dibayarkan.

Amortisasi utang untuk anuitas dimuka nilai sekarang

Hampir sama seperti amortisasi utang untuk anuitas biasa nilai sekarang.Angsuran pertama hanya untuk melunasi amortisasi utang karena belum ada biaya bunga yang timbul mengingat pembayaran pertama dilakukan pada hari ini.

Pelunasan pembayaran adalah pembayaran hutang hingga lunas atau selesai. Ketika memiliki uang berlebih, orang akan cenderung segera melunasi hutang yang dimilikinya.

30. Budi meminjam uang sebesar Rp8.700.000 kepada koperasi.pinjaman tersebut akan diangsur setiap bulan masing" sebesar Rp 150.000,Rp180.000,Rp210.000 dan seterusnya hingga pinjamannya lunas terbayar. Pada bulan berapakah pinjaman dapat lunas terbayar


a = 150.000
b = 210.000-180.000 = 30.000
Un = 150.000 + 30.000n - 30.000
Un = 30.000n + 120.000
U12 = 360.000 + 120.000 = 480.000

Sn = n/2 x ( a + Un )
8.700.000 = n/2 x ( 150.000 + 30.000n + 120.000)
8.700.000 x 2 = n x ( 270.000 + 30.000n )
30.000n² + 270.000n - 17.400.000 = 0 ( dibagi 30.000)
n² + 9n - 580 = 0
(n + 29)(n - 20) = 0
n = 20

maka pinjaman dapat lunas terbayar pada bulan ke-20
