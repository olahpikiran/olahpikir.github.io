6. besar sudut DOC adalah ​


3x + 18 + 5x + 18 = 180

8x + 36 = 180

8x = 180 - 36

8x = 144

x = 144/8

x = 18

Besar AOB = DOC

Besar DOC

3x + 18

3(18) + 18

54 + 18

72°

Jawaban:

72°

Penjelasan dengan langkah-langkah:

AOB = 3x + 18

AOD = 5x + 18

Sudut AOB + AOD = 180 (karena garis lurus)

3x + 18 + 5x + 18 = 180

8x + 36 = 180

8x = 180 - 36

8x = 144

x = 144 ÷ 8

x = 18

Sudut AOD = 5x + 18

Sudut AOD = 5 × 18 + 18

Sudut AOD = 90 + 18

Sudut AOD = 108

Sudut AOD + DOC = 180

108 + DOC = 180

DOC = 180 - 108

DOC = 72°

11. jika ada PORSENI di sekolah, apa absensi kelas berlaku?​


Jawaban:

Jika ada PORSENI di sekolah, maka absensi kelas masih berlaku. Siswa yang ikut serta dalam PORSENI masih harus hadir di kelas dan mengikuti proses belajar mengajar sesuai dengan jadwal yang telah ditentukan.

Namun, siswa yang terpilih untuk mengikuti PORSENI mungkin akan mendapatkan dispensasi untuk tidak hadir di kelas pada saat latihan dan mengikuti lomba, tergantung pada kebijakan sekolah masing-masing.

Penjelasan:

PORSENI adalah singkatan dari Perlombaan Olahraga Siswa Nasional, yaitu lomba olahraga tingkat nasional yang diadakan di sekolah-sekolah di Indonesia.

13. apa yang dimaksud DOC


Dalam teknologi komputasi, DOC atau doc (singkatan dari 'document' ) adalah sebuah ekstensi berkas untuk aplikasi pengolah kata yang terdapat di Microsoft Word Binary File Format.[4] Sejarah mengatakan bahwa ekstensi tersebut pernah digunakan pada teks biasa, khususnya program atau perangkat keras komputer di berbagai sistem operasi. Pada 1980an, WordPerfect menggunakan DOC sebagai format bawaan mereka. Kemudian, pada dasawarsa 1990an, Microsoftmemilih untuk menggunakan ekstensi DOC untuk format bawaaan Microsoft Word.
