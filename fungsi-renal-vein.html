2. apa fungsi korteks ,renal pelvis,medulla,renal arteri,renal vein,ureter?


Fungsi dari :
1) korteks => menyaring darah
2) renal pelvis => menampung urine (hasil proses penyaringan) sementara
3) medulla => tempat terjadinya reabsorbsi dan augmentasi fitrat urin
4) renal arteri => menghantarkan darah ke dalam ginjal untuk disaring di glomerulus
5) ureter => mengalirkan urin ke kandung kemih

korteks: transportasi hara dari epidermis ke dalam teras akar.

renal pelvis: sebagai saluran untuk mengalirkan urin dari ginjal menuju ke ureter.

medulla: untuk menahan massa jaringan yg disebut piramida ginjal disetiap ginjal.

renal arteri: membawa darah ke dalam ginjal untuk disaring di glomelurus.

renal vein: membawa darah keluar dari ginjal menuju vena cava inferior kemudian kmbali ke jantung 

ureter: mengantarkan urin dari ginjal menuju kantung kemih.
