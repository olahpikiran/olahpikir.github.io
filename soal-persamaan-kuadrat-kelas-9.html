titik puncak : {-1 , 6}

Penjelasan dengan langkah-langkah:

Titik puncak : {x , y}

x : sumbu simetri

y : nilai optimum

2x² + 4x + 8 = 0

a = 2 , b = 4 , c = 8

• sumbu simetri (x)

x = - b/2a

x = -4/2(2)

x = -4/4

x = -1

• nilai optimum (y)

y = - D / 4a

y = - (b² - 4ac) / 4a

y = - (4² - 4 × 2 × 8) / 4 × 2

y = - (16 - 64) / 8

y = - (- 48) / 8

y = 48/8

y = 6

Penyelesaian:

A=2

B=4

C=8

----------------------------------------

-b/2a

=-4/2(2)

=-4/4

=-1

----------------------------------------

D/4a

=b²-4ac/-4a

=4²-4(2)(8)/-4(2)

=16-8×8/-8

=16-64/-8

=-48/-8

=6

----------------------------------------

Jadi titik puncak nya ada di titik (-1,6)

14. Bantuin kk Soal Kelas 9 tentang persamaan Kuadrat


Jawaban:

Bentuk umum persamaan kuadrat adalah ax² + bx + c = 0. Dalam hal ini, a ≠ 0 dan a, b, dan c merupakan bilangan riil. Lebih lanjut, a ≠ 0 sebab jika a = 0, maka tidak akan tebentuk persamaan kuadrat.

Penjelasan:

Poin a.

[tex]\[\begin{gathered} 3x - {x^2} = - 2 \hfill \\ \Leftrightarrow 3x - {x^2} + 2 = 0 \hfill \\ \Leftrightarrow - {x^2} + 3x + 2 = 0 \hfill \\ \end{gathered} \][/tex]

Poin b.

[tex]\[\begin{gathered} 2{x^2} = - 5 + x \hfill \\ \Leftrightarrow 2{x^2} + 5 - x = 0 \hfill \\ \Leftrightarrow 2{x^2} - x + 5 = 0 \hfill \\ \end{gathered} \][/tex]

Poin c.

[tex]\[\begin{gathered} 2x\left( {2x + 2} \right) = 2x\left( {x - 3} \right) \hfill \\ \Leftrightarrow 4{x^2} + 4x = 2{x^2} - 6x \hfill \\ \Leftrightarrow 4{x^2} - 2{x^2} + 4x + 6x = 0 \hfill \\ \Leftrightarrow 2{x^2} + 10x = 0 \hfill \\ \end{gathered} \][/tex]

Poin d.

[tex]\[\begin{gathered} 2{x^2} - 5 = 3x \hfill \\ \Leftrightarrow 2{x^2} - 5 - 3x = 0 \hfill \\ \Leftrightarrow 2{x^2} - 3x - 5 = 0 \hfill \\ \end{gathered} \][/tex]

Poin e.
