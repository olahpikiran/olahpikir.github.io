Simple Present :

Digunakan untuk menyatakan rutinitas, turth fact, masa sekarang, dan frekuensi.

Form :

(+) S + V1 + O/C

(-) S + Do/Does + not + V1 + O/C

(?) Do/does + S + V1 + O/C + ?

Keterangan :

S = Subject ; I, you, they, we, she, he it.V1 = Infinitive, e. g : eat, go leave, etc.O = Object (Objek)C = Compliment (Keterangan)

Note :

Untuk do, digunakan untuk subjek I you, they, we.Untuk does, digunakan untuk subjek she, he, it.Untuk subjek tunggal,verb nya ditambah s/es.
