Pembahasan

Dimensi ialah penyimbolan aluas tata penulisan suatu besaran dengan menggunakan simbolisme dari besaran pokok. Terkait besaran, besaran dicacah menjadi 2 kriteria yaitu besaran pokok dan besaran turunan :

20. Dimensi besaran pokok dan turunan


Di dalam mekanika, besaran pokok panjang, massa, dan waktu merupakan besaran yang berdiri bebas satu sama lain, sehingga dapat berperan sebagai dimensi. Dimensi besaran panjang dinyatakan dalam L, besaran massa dalam M, dan besaran waktu dalam T. Persamaan yang dibentuk oleh besaran-besaran pokok tersebut haruslah konsisten secara dimensional, yaitu kedua dimensi pada kedua ruas harus sama. 21. Besaran pokok nama besaran satuan dan dimensi


Nama besaran pokok beserta satuan dan dimensinya !

PEMBAHASAN

Besaran merupakan sesuatu yang dapat diukur, serta memiliki nilai besar dalam satuan. Satuan digunakan sebagai pembanding dalam pengukuran. Satuan Internasional ( SI ) adalah satuan hasil konferensi para ilmuwan di Paris, yang membahas tentang berat dan ukuran suatu benda. Berdasarkan satuannya, besaran dibagi menjadi [tex]2[/tex] yaitu besaran pokok dan besaran turunan.

Besaran Pokok

Besaran pokok merupakan besaran yang satuannya telah ditentukan terlebih dahulu. Satuan besaran-besaran itu telah ditentukan sebagai acuan dari satuan besaran-besaran lain.

Nama besaran pokok beserta satuan dan dimensinya yaitu sebagai berikut :

1. Panjang

Satuan : Meter ( m )

Dimensi : [ L ]

Alat ukur : Mistar, jangka sorong, dan mikrometer sekrup

2. Massa

Satuan : Kilogram ( kg )

Dimensi : [ M ]

Alat ukur : Timbangan atau neraca

3. Waktu

Satuan : Sekon ( s )

Dimensi : [ T ]

Alat ukur : Jam tangan, jam dinding, dan stopwatch

4. Suhu

Satuan : Kelvin ( K )

Dimensi : [ Θ ]

Alat ukur : Termometer

5. Kuat arus

Satuan : Ampere ( A )

Dimensi : [ I ]

Alat ukur : Amperemeter dan multimeter

6. Intensitas cahaya

Satuan : Candela ( cd )

Dimensi : [ J ]

Alat ukur : Light meter / Lux meter, goniophotimeter, dan spektrofotometer

7. Jumlah zat

Satuan : Mol ( mol )

Dimensi : [ N ]

Alat ukur : Tidak ada alat khusus yang bisa langsung mengukur jumlah zat

Besaran Turunan

Besaran turunan merupakan besaran yang satuannya ditentukan dari penurunan satuan besaran-besaran pokok penyusunnya. Contoh besaran turunan yaitu usaha, gaya, energi, percepatan, kecepatan, luas, volume, daya, dan lain sebagainya.
