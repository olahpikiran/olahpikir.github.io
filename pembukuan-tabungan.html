Bilangan Positif ( + )Bilangan Negatif ( - )

[tex]\tt {\bold {\underline {Rumus\:Penjumlahan}}}[/tex]

➪Penjumlahan pada kancing ( + ) dan kancing ( - )

Catatan :

Pasangan ( + ) dan ( - ) menyatakan bilangan nol

Penjumlahan bilangan nol tidak berarti, artinya mengubah nilai (tetap ).

[tex]\tt {\bold {\underline {Rumus\:Pengurangan}}}[/tex]

Pengurangan bilangan bulat juga dapat dilakukan dengan beberapa cara seperti pada penjumlahan bilangan bulat.

➪Pengurangan pada kancing ( + ) dan ( - )

Sama seperti pada penjumlahan bilangaj bulat, simbol ( + )untuk bilangan positif ( - ) untuk bilangan negatif. Ingat kembali bahwa pasangan ( + ) dan ( - ) menyatakan bilangan nol.

Contoh

5 - 3 = 2

4 - 5 = 4 + 0 - 5 = -1

[tex]\tt {\bold {\underline {Rumus\:Perkalian}}}[/tex]

Cara mengerjakan bilangan bulat sama seperti perkalian pada bilangan cacah.

3 × 2 = 6 ➪Dikurang 2

2 × 2 = 4 ✎Dikurang 2

1 × 2 = 2 ➪Dikurang 2

0 × 2 = 0 ✎Dikurang 2

-1 × 2 = -2 ➪Dikurang 2

-2 × 2= -4 ✎Dikurang 2

-3 × 2 = -6 ➪Dikurang 2

Hasil perkalian dua bilangan bulat bertanda sama adalah bilangan positif.Bilangan positif dikali bilangan positif sama dengan bilangan positif.Bilangan negatif dikali bilangan negatif sama dengan bilangan positif.Hasil perkalian dua bilangan bulat berbeda tanda adalah bilangan bulat negatif.Bilangan positif dikali bilangan negatif sama dengan bilangan negatif.Bilangan negatif dikali bilangan positif sama dengan bilangan negatif.

Catatan Rumus

Positif × Positif = Positif

Negatif × Negatif = Positif

Positif × Negatif = Negatif

Negatif × Positif = Negatif

Dit = Berapakah perbandingan uang beli buku dan uang yang di tabung ?

Jawab =

Uang beli buku + Uang yang ditabung = Rp 15.000

Uang beli buku - Uang yang ditabung = Rp 3.000

Rp 15.000 - Rp 3.000 = Rp 12.000 .

Rp 12.000 : 2 = Rp 6.000 . ( uang yang ditabung )

Rp 15.000 - Rp 6.000 = Rp 9.000 . ( uang beli buku )

Jadi =

Uang beli buku = Rp 9.000,00

Uang yang ditabung = Rp 6.000,00
