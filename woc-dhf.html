13. Pada gambar diatas,DH // EG dan EJ // FH.Besar DEJ = 33 derajat dan DHF = 84 derajat. Tentukan nilai x+y!​


Jawaban:

159°

Penjelasan dengan langkah-langkah:

Besar Sudut JEG = GHJ = 84°

Mencari Besar Sudut y

Besar Sudut HJE = HGE (y) = (360 - 2 (84))/2 = (360 - 168)/2 = 192/2 = 96°

Mencari Besar Sudut x

Besar sudut EDJ (x) = 180 - (180 - 96) - 33 = 180 - 84 - 33 = 63°

Mencari Nilai x + y

x + y = 63 + 96 = 159°

29. tuliskan persamaan termokimia dari pernyataan berikut. a. DHf CaCo3OH= -1207 kj/mol b.HDc CH3OH= -638 kj/mol


Kelas         : XI
Pelajaran   : Kimia
Kategori     : Termokimia
Kata Kunci : persamaan termokimia, perubahan entalpi, pembentukan, pembakaran, standar

Jawaban
Persamaan termokimia
(a). Ca(s) + C(s) + 2O₂(g) + ¹/₂ H₂(g) → CaCO₃OH     ΔHf° = -1207 kJ/mol
(b). CH₃OH(l) + ³/₂ O₂(g) → CO₂(g) + 2H₂O(g)      ΔHc° = -638 kJ/mol
 
Pembahasan

(a). Notasi ΔHf° menyatakan entalpi pembentukan standar untuk pembentukan 1 mol senyawa dari zat-zat tunggal penyusunnya. ΔHf° dapat bernilai positif (endoterm) atau negatif (eksoterm).
⇒ ΔHf° bermakna 'standard enthalpy of formation'

(b). Notasi ΔHc° menyatakan entalpi pembakaran standar yaitu pembakaran 1 mol senyawa oleh oksigen pada keadaan standar. ΔHc° selalu negatif (eksoterm). 
⇒ ΔHc° bermakna 'standard enthalpy of combustion'
