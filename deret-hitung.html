6. perbedaan deret ukur dan deret hitung


Deret ukur
Adalah deret yang perubahan suku-sukunya berdasarkan perkalian terhadap suatu bilangan tertentu. Bilangan yang membedakan suku-suku sebuah deret ukur dinamakan pengganda, yakni hasil bagi nilai suatu suku terhadap suku lainnya.


Deret hitung
Adalah deret yang perubahan sukunya berdasarkan penjumlahan terhadap suatu bilangan tertentu. Bilangan yang membedakan suku-suku dari deret hitung ini dinamakan pembeda, yaitu selisih antara nilai-nilai suku yang berurutan.

8. contoh deret hitung dan deret ukur


Deret hitung=Contoh : 1,5,9,13

Suku ke-n

Sn : a + ( n – 1 ) b

a : suku pertama

b : pembeda

n : indeks suku

Deret ukur=Contoh : 2,4,8,16,32

Suku ke-n

Sn : a.p n – 1

Dimana a : suku pertama

             p : pengganda

             n : indeks suku

11. hitung deret tak hingga ada deret geometri berikut 16+8+4+...​


Jawab:

Penjelasan dengan langkah-langkah:

16 + 8 + 4 + ...

a = 16

r = 8/16 = ½

S~ = 16 / (1 - ½)

S~ = 16 / ½

S~ = 16 (2)

S~ = 32

S~ = jumlah tak hingga.

Jawab:

S∞ = 32

Penjelasan dengan langkah-langkah:

Rumus deret tak berhingga untuk suku genap

S∞(genap) = a/1-r

keterangan:

a = suku pertama (yaitu 16)

r = rasio (suku kedua (U2) ÷ suku pertama (U1) = 8/16 = 1/2

S∞ = a/1-r = 16/1-(1/2)

S∞ = 16 ÷ (1/2)

S∞ = 32

jadi, deret tak terhingganya adalah 32

29. Deret hitung A mempunyai nilai sebagai berikut : S2 = 8 pembeda (b) = 2. Deret hitung B : S3 = 28, b = -4. - Pada suku keberapa deret hitung A dan B nilainya sama. - Hitung J5 dari deret A dan B.


Deret Hitung

Rumus Suku ke-n (Sn)

Sn = a + (n - 1) b

a = suku pertama

b = pembeda

[tex]\\[/tex]

Rumus Jumlah n Suku (Jn)

Jn = [tex]\frac{n}{2}\\[/tex] × (a + Sn)

Jn = [tex]\frac{n}{2}\\[/tex] × (2a + (n - 1) b)

[tex]\\[/tex]

Diketahui:

Deret hitung A dengan S2 = 8; b = 2

S2 = a + (2 - 1) 2

8 = a + (1)2

8 = a + 2

a = 6

Rumus suku ke-n deret hitung A adalah

Sn = 6 + (n - 1) 2

Sn = 6 + 2n - 2

Sn = 2n + 4

[tex]\\[/tex]

Deret hitung B dengan S3 = 28; b = -4

S3 = a + (3 - 1)(-4)

28 = a + (2)(-4)

28 = a - 8

a = 36

Rumus suku ke-n deret hitung B adalah

Sn = 36 + (n - 1)(-4)

Sn = 36 - 4n + 4

Sn = 40 - 4n

[tex]\\[/tex]

a.)

Sn deret hitung A = Sn deret hitung B

2n + 4 = 40 - 4n

2n + 4n = 40 - 4

6n = 36

n = 6

Pada suku ke-6, deret hitung A dan B nilainya sama.

DH A: S6 = 2(6) + 4 = 12 + 4 = 16

DH B: S6 = 40 - 4(6) = 40 - 24 = 16

[tex]\\[/tex]

b.)

Hitung J5

Deret Hitung A:

Jn = [tex]\frac{n}{2}\\[/tex] × (a + Sn)

J5 = [tex]\frac{5}{2}\\[/tex] × (a + S5)

= [tex]\frac{5}{2}\\[/tex] × [6 + (2(5) + 4)]

= [tex]\frac{5}{2}\\[/tex] × (6 + 14)

= [tex]\frac{5}{2}\\[/tex] × 20

= 50

[tex]\\[/tex]

Deret Hitung B:

Jn = [tex]\frac{n}{2}\\[/tex] × (a + Sn)

J5 = [tex]\frac{5}{2}\\[/tex] × (a + S5)

= [tex]\frac{5}{2}\\[/tex] × [28 + (40 - 4(5)]

= [tex]\frac{5}{2}\\[/tex] × (28 + 20)

= [tex]\frac{5}{2}\\[/tex] × 48

= 120

[tex]\\[/tex]

Semoga membantu.
