3. Apa yang anda ketahui mengenai Long Short Term Memory (LSTM). Jelaskan dan gambarkan arsitekturnya


Long short term memory network (LSTM) merupakan sistem penyimpanan data yang dapat memproses, memprediksi, dan mengklasifikasikan informasi yang telah disimpan dalam jangka waktu yang cukup lama. LSTM merupakan bentuk lain dari recurrent neural network atau RNN.

Pembahasan:

Long term memory merupakan tempat penyimpanan informasi secara permanen dan banyak hal yang dapat bertahan dalam waktu lama pada memori ini. LSTM memiliki fungsi pada kasus peramalan (forecasting) yaitu menyusun perkiraan yang benar atau dapat dibuktikan terhadap suatu variabel. Prediksi terbaik didasarkan pada tingkat kesalahan prediksi, di mana semakin kecil tingkat kesalahan yang dihasilkan, semakin tepat sebuah metode dalam memprediksi.

Kredit Jangka Pendek ( Short-term loan )
Yaitu kredit yang jangka waktu pengembaliannya kurang dari satu tahun. Misalnya kredit untuk membiayai kelancaran operasi perusahaan, termasuk didalamnya berupa kredit modal kerja. Kredit jangka pendek dapat di urutkan dalam tiga kelompok, antara lain : (1)Kredit dagang (trade credit) antar perusahaan, (2)Pinjaman dari suatu perusahaan dagang, (3)Surat dagang.

sedangkan Kredit jangka panjang (Long-term loan)
Yaitu kredit yang jangka waktu pengembaliannya melebihi tiga tahun. Misalnya kredit investasi untuk membiayai suatu proyek dan perluasan usaha.

13. Perbedaan secondary memory dengan main memory


Secondary dan main memory adalah dua jenis perangkat memori yang terdapat dalam sebuah sistem komputer dan berfungsi menyimpan data.

Pembahasan

Pada kesempatan ini, soal meminta kita untuk menyajikan perbedaan dari kedua memori tersebut. Berikut kakak akan mencoba menjawabnya.

MAIN MEMORI atau memori utama adalah perangkat penyimpanan data pada komputer dan umumnya berbentuk RAM.

Memori utama berfungsi sebagai media penyimpanan data sementara yang kemudian akan diolah oleh unit processor untuk kemudian disampaikan kepada pengunakan melalui media output.

Sementara itu, SECONDARY MEMORY atau memori tambahan adalah perangkat penyimmpanan data jangka panjang dan biasanya berbentuk ROM.

Dalam praktiknya, memori tambahan juga kerap difungsikan untuk menyediakan ruang penyimpanan data tambahan ketika tidak tersedia ruang cukup di memori utama.
