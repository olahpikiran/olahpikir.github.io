6. Cara membuat output angka 5 berbentuk segitiga dengan bahasa C menggunakan borland C++​


Jawaban:

#include <stdio.h>

#include <conio.h>

main(){

int j,i,n;

printf("masukan angka : ");

scanf("%d",&n);

for(i=n;i>=1;i--){

for(j=1;j<=i;j++)

{

printf("%d",j);

}

printf("\n");

}

getch();

}

Penjelasan:

Penjelasan kode diatas :

– baris 1 dan 2 itu untuk memanggil library stdio.h dan conio.h, fungsinya agar dapat menjalankan perintah printf(), scanf(), getch() dll

– baris 5 membuat variabel j, i dan n dengan tipe data int

– baris 6 mencetak kata “masukan angka : ”

– baris 7 meminta inputan berupa angka yang nantinya akan dimasukan kedalam variabel n. karena n bertipe data int maka menggunakan %d, kalau char %c, kalau string %s, kalau float %f dll. untuk variabel didalam scanf() harus ditambah simbol “&” agar tidak error. contoh : &n , seperti kode diatas

– baris 8 itu perintah perulangan pertama yang nantinya kalau variabel i dicetak akan menhasilkan output : n n-1 n-2 sampai ketemu 1

– baris 9 itu perintah perulangan kedua yang nantinya kalau variabel j dicetak akan menhasilkan output : 1 2 3 sampai dengan i, misal n yang diinputkan 5, sudahterjadi 2 perulangan pada perulangan pertama maka nilai i adalah 3 dan nilai j adalah 123

– baris 11 untuk mencetak nilai j

– baris 13 mencetak enter. setelah perulangan kedua dijalankan akan mencetak enter

– baris 15 meminta inputan berupa charakter, tapi dalam program ini, perintah ini dipakai hanya untuk menahan output atau agar outputnya tetap muncul dilayar

Dan ini ss hasil / outputnya

10. Bagaimana cara membuat Program dan Aplikasi dengan bahasa C++ di DEV C++?? bukan di Borland..


- Jika belum terpasang program Dev C++nya, download terlebih dahulu dan install/pasang program tersebut 
- Selanjutnya klik 2x icon/shortcut Dev C++ yang ada di dekstop 
- Kemudian klik menu File, lalu pilih pilihan New, dan pilih Source File, atau bisa juga dengan menekan tombol Ctrl+N pada keyboard 
- Lalu input kode-kode programnya 
- Jika ingin dicompile, klik menu Execute, lalu tinggal pilih Compile, dan jika ingin mencoba aplikasi yang tadi dibuat tinggal pilih Run, atau kalau mau langsung mencoba aplikasi setelah dicompile maka pilih Compile and Run 
- Kalau ingin disimpan kodenya, klik File, kemudian pilih Save

15. Cara membuat output angka perkalian berbentuk segitiga dengan bahasa C menggunakan borland C++​


#include <iostream>

using namespace std;

int main()

{

int n;

cout << "Masukkan angka: ";

cin >> n;

for (int i = 1; i <= n; i++)

{

for (int j = 1; j <= i; j++)

{

cout << i * j << " ";

}

cout << endl;

}

return 0;

}

Program ini akan meminta pengguna untuk memasukkan sebuah angka, kemudian menampilkan perkalian dari angka tersebut dalam bentuk segitiga.

30. Cara menampilkan Toolbar di Borland C yang hilang gimana ya?


1. Keluar dari semua contoh Visio.
2. Klik mulai, klik Jalankan, ketik regedit, dan kemudian klik OK.

3. Klik subkunci registri berikut, yang sesuai untuk versi Visio yang sedang Anda Operasikan:

Microsoft Office Visio 2007

HKEY_CURRENT_USER\Software\Microsoft\Office\12.0\Visio\Toolbars

Microsoft Office Visio 2003

HKEY_CURRENT_USER\Software\Microsoft\Office\11.0\Visio\Toolbars

Microsoft Visio 2002

HKEY_CURRENT_USER\Software\Microsoft\Visio\Toolbars

Microsoft Visio 2000

HKEY_Current_User\Software\Visio\Visio 2000\Toolbars

4. Pada Edit menu, klik Hapus. Subkunci Toolbar akan dihapus.

5. Jika Anda diminta untuk mengkonfirmasi penghapusan, klik ya.

6. Pada menu registri , klik keluar.

Peninjau suntingan registri secara otomatis menyimpan perubahan.

Ketika Anda me-restart Visio, Visio kembali menciptakan subkunci dihapus dan mengembalikan Toolbar dan menu bar lokasi default mereka.
