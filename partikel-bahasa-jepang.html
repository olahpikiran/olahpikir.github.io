1. apa saja partikel partikel yang ada di bahasa Jepang? Jelaskan​


Jawaban :

Partikel-partikel yang ada dalam Bahasa Jepang beberapanya seperti partikel Wa, partikel No, partikel De, partikel Ni, partikel E, partikel Wo/O, partikel To, partikel Ga, partikel Kara, partikel Made, dll.

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Pengertian Partikel Bahasa Jepang

Partikel dalam Bahasa Jepang merupakan sebuah kata bantu kalimat, yang jumlahnya ada banyak dan memiliki fungsinya tersendiri. Kalau dalam Bahasa Jepang, partikel ini disebut dengan 助詞 (Joshi). Joshi dibagi menjadi 4 jenis kelompok, yakni sebagai berikut.

格助詞 (Kakujoshi) : P. Penanda kasus接続助詞 (Setsuzokujoshi) : P. Konjungsi副助詞 (Fukujoshi) : P. Keterangan終助詞 (Shuujoshi) : P. Akhir kalimat

Contoh-Contoh Partikel Bahasa Jepang

1. Partikel は (Wa)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi sebagai penanda subjek atau topik dari suatu kalimat. Kadang bisa diartikan sebagai "Adalah". Ditulis dengan huruf Ha, tapi dibaca Wa.

2. Partikel の (No)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menyatakan suatu kepemilikan. Karena itu partikel ini kadang bisa berarti "Milik", "Punya", "Nya", dll.

3. Partikel で (De)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan tempat melakukan suatu aktivitas, menunjukkan keterangan cara/alat, menyatakan sebab atau alasan, dan lainnya. Kadang bisa berarti "Di", "Dengan", "Karena", dll.

4. Partikel に (Ni)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan letak atau tempat keberadaan sesuatu, menunjukkan arah atau tujuan, menunjukkan waktu, dan lainnya. Biasa diartikan sebagai "Di", "Ke", "Pada", dll.

5. Partikel へ (E)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan arah dan orang yang dituju. Partikel ini memiliki arti "Ke" atau "Untuk". Ditulis dengan huruf He, tapi dibaca E.

6. Partikel を (Wo / O)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi sebagai penanda objek atau kata benda. Tidak memiliki arti. Partikel ini biasa diletakkan di tengah kata benda dan kata kerja. Ditulis dengan huruf Wo, tapi dibaca O (dalam Romaji bisa ditulis Wo maupun O).

7. Partikel と (To)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menyebutkan hal yang lebih dari satu, menunjukkan kata yang dimaksud sebagai lawan yang diajak, menunjukkan keterangan hal yang terjadi setelahnya, dll. Kadang bisa berarti "Dan", "Dengan", "Bersama", "Lalu", dll.

8. Partikel が (Ga)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menegaskan subjek dari suatu kalimat, menyambungkan kalimat yang berlawanan, dll. Biasanya tidak memiliki arti, namun saat menyambungkan kalimat yang berlawanan dapat berarti "Tapi".

9. Partikel から (Kara)

Termasuk ke dalam kelompok Kakujoshi. Berfungsi untuk menunjukkan titik awal dari sesuatu, dan menyatakan sebab atau alasan. Bisa berarti "Dari", "Karena", dll.

10. Partikel まで (Made)

Termasuk ke dalam kelompok Fukujoshi. Berfungsi untuk menunjukkan titik akhir dari sesuatu, dan menyatakan batasan yang lebih luas. Hampir selalu berarti "Sampai". Partikel ini kadang berpasangan dengan partikel Kara.

Sebenarnya masih ada banyak lagi partikel dalam Bahasa Jepang, tapi tidak bisa saya sebutkan semua karena tidak akan muat penjelasannya. Partikel-partikel yang saya sebutkan di atas hampir semuanya termasuk ke dalam kelompok Kakujoshi, karena memang kelompok Kakujoshi kebanyakan berisi partikel-partikel dasar dan penting untuk dihafal.

2. Apa saja partikel yang ada dalam bahasa jepang?


Pendahuluan

Partikel atau Joshi merupakan sebuah kata bantu kalimat yang memiliki beragam fungsi. Jumlah partikel Bahasa Jepang ada sekitar 188 partikel, dan masing-masing memiliki fungsinya tersendiri.

Partikel dibagi menjadi 4 jenis :

Kakujoshi : Partikel kasusSetsuzokujoshi : Partikel konjungsiFukujoshi : Partikel adverbialShuujoshi : Partikel akhir kalimat

Beberapa contoh partikel seperti partikel Wa, No, Ni, Ka, O, De, To, Kara, Made, Mo, dll. Saya akan menjelaskan sedikit beberapa partikel Bahasa Jepang beserta fungsinya.

--

Pembahasan

1). Partikel は (Wa)

Partikel ini ditulis dengan Hiragana 'Ha', tapi dibaca 'Wa'. Fungsinya yaitu sebagai penanda topik kalimat. Dalam beberapa situasi, partikel ini juga dapat diartikan sebagai "adalah".

2). Partikel の (No)

Partikel ini memiliki banyak fungsi, tapi fungsi utama nya yaitu untuk menyatakan suatu kepemilikan. Misalkan pada kalimat 'A no B', maka dalam Bahasa Indonesia artinya seperti "B (milik) A".

3). Partikel に (Ni)

Partikel ini juga memiliki banyak fungsi. Beberapa nya yaitu untuk menunjukkan waktu, letak/keberadaan sesuatu, tempat tujuan, dll.

4). Partikel か (Ka)

Partikel ini fungsi utamanya untuk membuat kalimat tanya. Biasa diletakkan di akhir kalimat, setelah kata 'desu'.

5). Partikel を (O)

Partikel ini ditulis dengan Hiragana 'Wo', tapi dibaca 'O'. Partikel ini berfungsi sebagai penanda objek. Biasa diletakkan di antara kata kerja dan kata benda atau objek.

6). Partikel で (De)

Partikel ini fungsinya beragam. Beberapa nya yaitu untuk menunjukkan tempat melakukan aktivitas, menyatakan keterangan cara/alat, menyatakan penyebab/alasan, sebagai penyambung kalimat, dll.

7). Partikel と (To)

Partikel ini memilki beragam fungsi, salah satunya sebagai kata hubung "dan" untuk kata yang kedudukannya setara. Misalkan pada kalimat 'A to B', maka dalam Bahasa Indonesia artinya seperti "A dan B".

8). Partikel から (Kara)

Partikel ini salah satu fungsinya untuk menunjukkan titik awal. Kalau dalam Bahasa Indonesia artinya seperti "dari". Partikel ini biasa dipasangkan dengan partikel Made.

9). Partikel まで (Made)

Partikel ini berfungsi untuk menunjukkan titik akhir. Kalau dalam Bahasa Indonesia artinya seperti "sampai". Partikel ini biasa dipasangkan dengan partikel Kara. Misalkan pada kalimat 'A kara B made', maka dalam Bahasa Indonesia artinya seperti "dari A sampai B".

10). Partikel も (Mo)

Partikel ini berfungsi untuk menyampaikan sesuatu yang bermakna setara. Kalau dalam Bahasa Indonesia artinya seperti "juga". Misalkan pada kalimat 'A mo B', maka dalam Bahasa Indonesia artinya seperti "A juga B".

6. sebutkan partikel yang berfungsi sebagai to be dalam bahasa Jepang.​


Jawaban:

Dalam bahasa Jepang, partikel disebut Joshi/Kakujoshi dan merupakan kata singkat dalam tata bahasa Jepang yang mengindikasikan hubungan antar kata dalam suatu kalimat. Partikel bahasa Jepang sebagaimana dengan preposisi berguna untuk menunjukkan waktu, arah, tempat, dan sebagainya.

1. Partikel は (wa)

menunjukkan topik atau subjek yang ingin dibicarakan dalam suatu kalimat.

Rumus: [topik kalimat] + [は/wa] + keterangan

2. Partikel も (mo)

Partikel ini digunakan untuk mengekspresikan ‘juga’, jika dialihbahasakan ke dalam bahasa Indonesia.

Rumus: [topik kalimat] + [も/mo] + keterangan

3. Partikel か (ka)

Partikel ini secara umum memiliki arti ‘kah?’ atau ‘apakah?’. Partikel ini digunakan untuk mengakhiri kalimat pertanyaan.

Rumus: [kalimat] + keterangan + [か/ka]

4. Partikel が (ga)

Partikel ini digunakan untuk menunjukkan subjek dari suatu kalimat ketika subjek tersebut adalah informasi baru bagi komunikan atau ketika dibutuhkan penekanan atas subjek pada kalimat.

Rumus: [subjek] + [が/ga] + kalimat lanjutan

5. Partikel を (wo)

Partikel ini berfungsi sebagai penghubung antara objek dan kata kerja.

Rumus: [objek] + [を/wo] + kata kerja

6. Partikel に (ni)

Partikel ini berfungsi untuk mengindikasikan lokasi tujuan atau posisi dari perpindahan pelaku atau benda.

7. Partikel の (no)

Digunakan untuk mengekspresikan kepunyaan atau penghubung antara dua kata benda. Penggunaan の (no) juga termasuk pada kasus relasi atau hubungan ganda.

8. Partikel へ (e)

Kegunaan partikel へ (e) hampir serupa dengan kegunaan partikel に (ni). Perbedaan antara keduanya terletak pada presisi dari posisi yang dideskripsikan dalam kalimat. Partikel へ (e) digunakan pada situasi ketika presisi dari lokasi tujuan tidak terlalu tepat.

9. Partikel で (de)

Partikel yang satu ini memiliki banyak penggunaan yang masing-masing mempunyai konteks yang berbeda. Namun, penggunaan partikel で (de) paling lazim digunakan untuk menunjukkan tempat aktivitas, cara, bahan, dan jumlah.

10. Partikel から (kara)

Partikel ini digunakan untuk mendeskripsikan titik mula dari sesuatu (waktu/tempat).

13. Quisss .....Sebutkan Partikel Partikel yang ada dalam bahasa jepang ....​

格助詞 (Kakujoshi) : Partikel kasus接続助詞 (Setsuzokujoshi) : Partikel konjungsi副助詞 (Fukujoshi) : Partikel adverbial終助詞 (Shuujoshi) : Partikel akhir kalimat

15. Kegunaan Partikel ( か ) dalam bahasa jepang?..​


Jawaban:

Partikel “ka” merupakan salah satu partikel yang umumnya digunakan di akhir kalimat dalam bahasa Jepang, yang berfungsi untuk menyampaikan suatu pertanyaan. Namun selain diungkapkan di akhir kalimat, dalam beberapa kasus, partikel “ka” juga bisa digunakan di tengah kalimat.

Cara penulisan partikel " Ka " :

1. Diletakkan pd akhir kalimat digunakan untuk bertanya.Contoh :

⇒Anata wa donata desu ka.

あなたはどなたですか。

Kamu siapa?

⇒Kore wa nan desu ka.

これは何ですか。

Ini apa?

2. Diletakkan di belakang kata tanya untuk menunjukkan sesuatu yg tidak jelas/tentu

Contoh :

⇒Poribukuro no naka ni nanika arimasu.

ポリ袋の中に何かあります。

Di dalam kantong plastik ada sesuatu.

⇒Itsuka tenki no ii hi ni tsuri ni ikitai desu.

いつか天気のいい日につりに行きたいです。

Kapan-kapan pada hari yg bercuaca baik saya ingin pergi memancing.

3. Digunakan dua hal atau lebih untuk menunjukkan salah satu. 'atau'

Contoh :

⇒Suika ka meron ka papaiya no you na

kudamono o kudasai.

すいかかメロンかパパイヤのような果物を

ください。

Berikan saya buah seperti semangka atau melon atau pepaya.

Sangat mudah untuk membentuk kalimat tanya dalam bahasa Jepang. Cukup menambahkan  か (ka) ke akhir kalimat sudah cukup.

か untuk Alternatif

bagaimana mereka menandai dua atau lebih kata sebagai alternatif. Ini seperti menggunakan "baik ... atau ..."

は (wa) , を (o) ,も (mo) ,に (ni) ,へ (e) , で (de) ,が (ga) , と (to)  , dan lain-lain.

Penjelasan:

は (wa)

mengikuti topik yang ingin dibicarakan oleh pembicara. Oleh karena itu, wa (dibaca ha) sering disebut partikel penanda topik. “Topik” seringkali merupakan subjek gramatikal, tetapi dapat berupa apa saja (termasuk objek tata bahasa, dan terkadang) kata kerja), dan mungkin juga mengikuti beberapa partikel lainnya.

pola kalimat = [ A ] wa [ B ] desu.

を (o)

menandai objek tata bahasa dari sebuah kalimat. Ini mengikuti kata benda dan frase kata benda.

pola kalimat = Kata benda + o (を) + kata kerja

も (mo)

berfungsi sebagai "juga" atau "terlalu" dalam . Ini digunakan untuk menunjukkan bahwa sesuatu yang sebelumnya telah dinyatakan juga berlaku untuk item yang saat ini sedang dibahas. Ini menggantikan ga, wa atau o saat digunakan.

pola kalimat= [Objek1] は [properti1 / tindakan1] で す

[Objek2] も [properti1 / tindakan1] で す.

に (ni)

menunjukkan tempat menuju di mana seseorang atau sesuatu bergerak. Ini diawali dengan nama tempat dan diikuti dengan kata kerja yang menunjukkan tindakan bergerak.

pola kalimat= tempat + に (ni)

へ (e)

pada dasarnya sama dengan に, kecuali ini menekankan arah daripada kedatangan. Perbedaan utamanya adalah penggunaan.へ tidak pernah digunakan sebagai "dari", "oleh", "di".

Selain itu, partikel の dapat mengikuti partikel へ secara langsung, sedangkan partikel tidak dapat mengikuti に.

pola kalimat = lokasi / arah + e (へ)

で (de)

digunakan untuk menunjukkan lokasi aksi. Pelaku tindakan (subjek) diikuti oleh salah satu partikel wa atau ga, dan tempat diikuti oleh de.

pola kalimat = lokasi + de

が (ga)

menandai subjek gramatikal kalimat saat pertama kali diperkenalkan ke percakapan.

と (to)

digunakan untuk menggabungkan kata benda menjadi daftar lengkap yang berfungsi sebagai kata benda tunggal: "dengan", "dan". Ini juga merupakan partikel yang digunakan untuk menunjukkan kutipan langsung (dari pikiran atau ucapan seseorang), berfungsi seperti tanda kutip.

pola kalimat = Kata benda + to (と) + Kata benda

の (no)

menunjukkan kepemilikan . Dalam struktur A no B, B milik A, namun, banyak kata benda bertindak seperti kata sifat jika diikuti oleh no.

Ini secara langsung mengikuti kata benda dan frase kata benda.

pola kalimat = Objek1 + の + Objek2

29. Partikel ( no) digunakan dalam kalimat bahasa Jepang pada saat apa saja?


Pembahasan

Partikel の (no)

Partikel ini merupakan salah satu huruf dari huruf hiragana. Partikel の (no) memiliki banyak fungsi tergantung konteks atau penggunaannya.

Fungsi-fungsinya yaitu sebagai berikut ini :

1). Berguna sebagai menyatakan suatu kepemilikan atau kepunyaan

Contohnya.

私の夫 (watashi no otto) : suami saya.俺の弟 (ore no otouto) : adik (lk) gw.

~~

2). Penghubung untuk kata benda

Untuk menghubungkan ke kata benda, tentunya memiliki pola sebagai berikut :

kt. benda 1 + partikel の + kt. benda 2 -> hal ini bertujuan agar kt. benda 1 memiliki fungsi untuk menjelaskan kt. benda yang ke-2.

Contohnya.

英語の先生 (eigo no sensei) : guru bahasa Inggris.彼は私のヒーローです (kare wa hiiroo desu) : dia (lk) pahlawanku.

~~

3). Sebagai bentuk kata tanya atau penanda pada pertanyaan

Umumnya menggunakan partikel の (no) ini bisa digunakan sebagai jenis pertanyaan dengan bentuk yang informal.

Contohnya.

どうしたの? (dou sh!ta no?) : ada apa?そこに行かないの? (soko ni ikanai no?) : gak pergi kesana?

~~

4). Menjelaskan suatu lokasi

Sebagai polanya untuk menjelaskan suatu lokasi adalah : lokasi yang besar + partikel の (no) + tempat bagian dari lokasi besar.

Contohnya.

インドネシアのバンドン (indoneshia no bandon) : Bandung, Indonesia.マレーシアのクアラルンプール (mareeshia no kuararunpuuru) : Kuala Lumpur, Malaysia.

~~

5). Menjadikan kata kerja ke kata benda

Memiliki polanya sebagai berikut : kt. kerja + partikel の (no) + kt. keterangan/partikel lain.

Contohnya.

私は書くのが大好きです (watashi wa kaku no daisuki desu) : aku suka menulis.私は教えるのが好きです (watashi wa oshieru no ga suki desu) : aku senang mengajar.

~~

6). Sebagai penegasan suatu kalimat

Untuk polanya yaitu : kalimat + partikel の (no).

Contohnya adalah >

お金がないの! (okaneganai no!) : aku gak ada duit.好きなのよ (suki na no yo) : aku suka.

~~

7). Menjelaskan mengenai dimana suatu buatan berasal

Contohnya.

インドネシアの服 (indoneshia no fuku) : baju buatan Indonesia.インドネシアの人形 (indoneshia no ningyou) : boneka buatan Indonesia.

~~
