4. Apa perbedaan dari MAD, MSE dan MAPE? Dalam kondisi apa/kapan MAD, MSE dan MAPE digunakan?


Jawaban:

MAD (Mean Absolute Deviation = Rata- rata Penyimpangan Absolut), MAPE (Mean Absolute Percentage Error = Rata- rata Persentase Kesalahan Absolut), dan MSE (Mean Square Error = Rata-rata Kuadrat Kesalahan).

digunakan saat:

1.MAPE (Mean Absolute Percent Error) digunakan jika ukuran variabel peramalan merupakan faktor penting dalam mengevaluasi akurasi peramalan tersebut. MAPE memberikan petunjuk seberapa besar kesalahan peramalan dibandingkan dengan nilai sebenarnya dari series tersebut

2. Mean Squared Error (MSE) adalah metode lain untuk mengevaluasi metode peramalan. Masing-masing kesalahan atau sisa dikuadratkan. Kemudian dijumlahkan dan ditambahkan dengan jumlah observasi. Pendekatan ini mengatur kesalahan peramalan yang besar karena kesalahan-kesalahan itu dikuadratkan.

8. mise o yatteimasu artinya​


mise o yatte imasu artinya dalam bahasa Indonesia adalah berbisnis di toko.

Pembahasan

店おやっています

mise o yatte imasu

Artinya :

Berbisnis di toko.Menjalankan toko.

________

Untuk menyatakan suatu pekerjaan atau profesi dalam wirausaha, maka dapat dinyatakan dengan pola sebagai berikut dalam bahasa Jepang yaitu :

KB (jenis kerja di bidang wirausaha) + partikel お (o) + やっています (yatte imasu).

________

店 (mise)

Artinya ialah bisa toko, warung, kedai, atau gerai. Versi hiragananya yaitu みせ

お (o)

Terdapatnya partikel お (o) dalam kalimat tersebut karena terdapat salah satu kata kerja yaitu やっています (yatte imasu), maka salah satu fungsi partikel お (o) tersebut ialah untuk kata bantu pada kata kerja.

やっています (yatte imasu)

Kata やってい (yatte) berasal dari kata やる (yaru), jadi kata やってい (yatte) tersebut merupakan kata kerja bentuk -te golongan l dan dalam bahasa Indonesia diartikan melakukan (tergantung konteks). Kemudian adanya penambahan kata います (imasu) ini mengubah kata kerja menjadi bentuk kata kerja present (sedang terjadi). Pola kalimatnya adalah :

KK -te + います (imasu).

Sehingga arti dari kalimat やっています (yatte imasu) tersebut adalah sedang melakukan, namun dalam kalimat diatas tersebut lebih baik diartikan (melakukan) berbisnis.

'Mise o yatte imasu' artinya "(Saya) menjalankan bisnis kedai/warung".

| | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | | |

Pembahasan :

Pola Kalimat :

== [KB](jenis usaha) o yatte imasu

== Menjalankan (bisnis) [KB](jenis usaha)

Digunakan untuk menjelaskan pekerjaan wirausaha yang dimiliki. Untuk pekerjaan-pekerjaan selain wirausaha, tidak bisa menggunakan pola ini.

Contoh Kalimat :

Kissaten o yatte imasu (Menjalankan bisnis kafe)Resutoran o yatte imasu (Menjalankan bisnis restoran)Mise o yatte imasu (Menjalankan bisnis kedai/warung)

- - - - -

MISE「店・みせ」

Berarti "Kedai", "Warung", "Toko", dsb. Kanji dari kata ini dibaca secara Kunyomi (cara baca Jepang), maka dari itu dapat ditambahkan huruf O di awalnya, jadi 'Omise'. Fungsi menambahkan huruf O di awal kata benda ini adalah untuk membuatnya lebih sopan.

O / WO「を」

Tidak ada arti. Kata ini merupakan partikel atau kata bantu kalimat untuk menandakan objek. Biasa diletakkan di antara kata benda dan kata kerja. Partikel ini dalam Hiragana ditulis dengan huruf を (O / Wo), dalam Romaji bisa ditulis O atau Wo, tapi dibacanya adalah O.

YATTE IMASU「やっています」

Berarti "(Sedang) melakukan". Tapi kalau dalam konteks kalimat tersebut, artinya lebih seperti "(Sedang) menjalankan (bisnis)". Kata ini merupakan salah satu hasil konjugasi dari kata kerja 'Yaru' yang artinya "Melakukan". Kata ini memiliki arti yang sama dengan 'Suru', tapi kedua kata ini tidak selalu bisa ditukar.

30. line muse atau marga muse di rp artinya apa y kak
