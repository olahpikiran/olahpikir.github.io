Cl - + H2SO4 = HSO4 - + HCl

2NaCl + H2SO4  Na2SO4 + 2HCl

1. Diketahui

mz = 180 grammp = 500 gram = 0.5 kg

ditanyakan molalitas =...?

Penyelesaian :

Mr glukosa (C₆H₁₂O₆) = 6 x Ar C + 12 x Ar H + 6 x Ar O

Mr glukosa = 6 x 12 + 12 + 6 x 16 = 180 g/mol

(Cek Tabel Periodik)

molalitas = mol glukosa/mp = (mz/ Mr) / mp = (180/180) / 0.5 = 0.5 molal

2. Diketahui

N Ca(OH)₂ = 0.2 NV = 250 mL

ditanyakan massa Ca(OH)₂ ?

Mr Ca(OH)₂ = Ar Ca + 2 x (Ar O + Ar H) = 40 + 2 x (16 + 1) = 74 g/mol

Molaritas Ca(OH)₂ = N : Valensi OH⁻ = 0.2 : 2 = 0.1 M

Menghitung Massa Ca(OH)₂

m = M x V x Mr = 0.1 M x 0.25 L x 74 g/mol = 1.85 gram

3. Diketahui :

Mr H₂SO₄ = 98 g/mol (cek tabel periodik)% massa = 98%massa jenis = 1.25 g/mL

Molaritas asam sulfat = ?

[asam sulfat] = mol x V = (massa asam sulfat/Mr) x Vtotal

diketahui : massa asam sulfat/Vtotal = massa jenis x %massa

[asam sulfat] = massa jenis x %massa : Mr

[asam sulfat] = 1.25 (g/mL) x 98/100 : 98 (g/mol) = 0.0125 mol/mL

Konversi satuan mL ke L

[asam sulfat] = 0.0125 mol/mL x 1000 mL/1 L = 12.5 mol/L

[asam sulfat] = 12.5 M

4. Pengenceran :

V2 = 500 mLM2 = 0.2 MM1 sesuai soal nomor 3 = 12.5 M

Ditanya cara membuat larutan ke-2 : ?

Dihitung volume larutan 12.5 M asam sulfat :

V1 x M1 = V2 x M2

V1 x 12.5 = 500 x 0.2

V1 = 8 mL

Artinya untuk membuat larutan 500 mL asam sulfat 0.2 M, adalah dengan menambahkan 8 mL larutan asam sulfat 98% ke dalam 498 mL aquadest.
