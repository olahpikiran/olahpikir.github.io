15. ubahlah bentuk bentuk pangkat negatif berikut kedalam bentuk pangkat positif!​


Bentuk pangkat positif dari pangkat negatif berikut adalah

[tex]a. \: {2}^{ - 5} = \frac{1}{ {2}^{5} } \\ \\ b. \: \frac{1}{ {2}^{ - 4} } = {2}^{4} [/tex]

Pembahasan:

Bilangan berpangkat

Bilangan berpangkat adalah bilangan eksponen yang biasa ditulis dengan [tex] a^{b} [/tex], dengan a dan b adalah bilangan bulat. a adalah bilangan pokok dan b adalah bilangan eksponen (pangkat).

Sifat-sifat bilangan berpangkat bulat positif

[tex] 1. a^{m} \times a ^{n} = a ^{(m+n)} \\ \\ 2. \frac {a^{m}}{a^{n}} = a^{m-n} \\ \\ 3. {(a^{m})}^{n} = a ^{m \times n} \\ \\ 4. {(a \times b)}^{m} = a^{m} \times b^{m} \\ \\ 5. {(\frac {a}{b}) }^{m} = \frac {a^{m}}{{b^m}} [/tex]

Dalam operasi hitung bilangan berpangkat, berpatokan pada sifat-sifat bilangan berpangkat diatas, untuk mempermudah mengingatnya, silakan perhatikan tips berikut:

Dengan syarat bilangan pokoknya sama maka jika

- dikali ➡ pangkatnya ditambahkan

- dibagi ➡ pangkatnya dikurangkan

- dipangkatkan ➡ pangkatnya dikalikan

- ditarik akar ➡ pangkatnya dibagi

Diketahui:

[tex]a. \: {2}^{ - 5} \\ b. \: \frac{1}{ {2}^{ - 4} } [/tex]

Ditanya:

Bentuk pangkat positif

Jawab:

Ingat sifat bilangan berpangkat :

[tex]\frac {a^{m}}{a^{n}} = a^{m-n} [/tex]

maka

[tex]a. \: {2}^{ - 5} = \frac{1}{ {2}^{5} } \\ \\ b. \: \frac{1}{ {2}^{ - 4} } \\ = ( {2}^{ - 4} ) {}^{ - 1} \\ = {2}^{ - 4 \times - 1} \\ = {2}^{4} [/tex]

Kesimpulan:

[tex]a. \: {2}^{ - 5} = \frac{1}{ {2}^{5} } \\ \\ b. \: \frac{1}{ {2}^{ - 4} } = {2}^{4} [/tex]

18. tuliskan bentuk W pangkat 3 × W pangkat 4 ke dalam bentuk perpangkatan paling sederhana. berapakah hasilnya? apakah kamu juga dapat menyederhanakannya bentuk w pangkat 3 × n pangkat 4 ? jelaskan jawabanmu


Tuliskan bentuk W pangkat 3 × W pangkat 4 ke dalam bentuk perpangkatan paling sederhana. berapakah hasilnya?

Hasilnya adalah  w⁷.

Apakah kamu juga dapat menyederhanakannya bentuk w pangkat 3 × n pangkat 4 ?

Hasil penyederhanaan dari w³ x n⁴ adalah n(wn)³.

Pembahasan

Untuk menyelesaikan soal-soal bentuk perpangkatan seperti ini, kita harus ingat sifat-sifat dari perpangkatan.

Berikut ini saya sertakan sifat-sifat bentuk perpangkatan.

Sifat-sifat perpangkatan

aᵇ x aⁿ = aᵇ⁺ⁿ aᵇ : aⁿ = aᵇ⁻ⁿ ( aᵇ)ⁿ = a ᵇ ˣ ⁿ (a/b)ⁿ = aⁿ/ bⁿ a ⁻ⁿ = 1 / aⁿ (ab)ⁿ = aⁿ x bⁿ

Berdasarkan bentuk perpangkatan diatas, mari kita selesaikan soal:

w³ x w⁴ berdasarkan sifat aᵇ x aⁿ = aᵃ⁺ⁿ, maka

w³ x w⁴ = w³⁺⁴

            = w⁷

w³ x n⁴ berdasarkan sifat (ab)ⁿ = aⁿ x bⁿ, maka

w³ x n⁴ = w³ x n³ x n

            = n (w³ x n³)

            = n(wn)³
