ANALYTICAL EXPOSITION TEXT

~ Genre teks ini menjabarkan pandangan penulis terhadap suatu masalah yang dianggap penting untuk didiskusikan.

The social function / communicative purpose  

~To convince the readers that there is an important thing in the case.

( untuk meyakinkan para pembaca bahwa ada hal penting dalam suatu kasus  )

Generic structure  

1. Thesis  ( general statement )

Paragraf pertama ini menjabarkan rumusan masalah yang didiskusikan berdasarkan cara pandang si penulis .

2. Arguments

Paragraf selanjutnya menjabarkan opini-opini si penulis  beserta didukung informasi  realistis ,  pembuktian fakta guna mendukung argumentasinya.

3. Reiteration

Paragraf akhir berisi kesimpulan / pengulasan kembali  / penekanan dari tujuan si penulis yang telah dipaparkan pada thesis nya.

Language features  ( Unsur kebahasaan )

~ Using internal conjunction to elaborate the argument ( firstly, secondly, ....... )

~  Using temporal conjunction  ( relative clause ) (  on the other hand , meanwhile.......)

~ Using -If conditional sentence

~ Using contrastive conjunction ( but, however .....)

~ Using causal conjunction ( because.........., caused by ....)

~ Using simple present tense  

Karena berdasarkan fakta kebenaran, tenses keseluruhan menggunakan simple present tense.

~ Using passive voice ( tindakan aksi difokuskan utama )

~ Using mental  expression words  ( lebih ke opini si penulis )

~ Using the abstract noun ( we,   they,  passive smokers........)

~ Using modal verbs     ( should........can........)

PEMBAHASAN:

Text di atas adalah Hortatory exposition yang mana pada akhir tulisan si penulis merekomendasikan sarannya untuk tidak mengijinkan siapapun merokok di restoran manapun.

Hortatory Exposition Text

Hortatory exposition is a text to persuade the reader or the listener that something should or should not be the case. Dengan kata lain, hortatory exposition merupakan sebuah teks yang berfungsi untuk memberikan argumen terhadap suatu hal dan dengan tujuan mempengaruhi atau memberikan persepsi baru kepada pembaca maupun pendengar.

Generic sructure of Hortatory Exposition text

Thesis : Thesis merupakan sebuah paragraf yang berfungsi untuk memaparkan akan suatu hal yag terjadi. mengenalkan masalah atau tema apa yang aka diangkat. Arguments : Pada bagian ini penulis mulai menyampaikan beberapa argumen ataupun alasan mengenai permasalahan atau tema yang diambil. Recommendation : Berisi rekomendasi, disini penulis atau penutur mulai meyakinkan si pembaca terhadap sudut pandang nya.

The Characteristic of Hortatory Exposition text

Menggunakan simple present tense Setiap paragraf berisi argumen terhadap sesuatuyang di bicarakan Setiap paragraf ditandai dengan kata firstly, secondly dan lain sebagainya. Pada parafgraf terdapat beberapa bahasa yang berfungsi sebagai bujukan seperti shoudly, should dan lainnya.
