22. tentukan titik kritis dari g(x) = x⁴ + x² + 3 dan gunakan pengujian pilihan anda sendiri dalam menentukan titik kritis tersebut memberikan nilai maksimum lokal atau minimum lokal dan sebutkan mana titik maksimum lokal dan mana titik minimum lokal tersebut. ​


Titik kritisnya adalah memberikan nilai minimum lokal dan titik tersebut adalah (0, 3).

Penjelasan dengan langkah-langkah:

Diketahui:

[tex]g(x)=x^4+x^2+3[/tex]

Ditanya:

Termasuk jenis apakah titik kritisnya?

Pembahasan:

Syarat titik kritis/stasioner adalah [tex]f'(x)=0[/tex]

Menentukan turunan pertama g(x)

[tex]g(x)=x^4+x^2+3\\g'(x)=4x^3+2x[/tex]

Menentukan turunan kedua g(x)

[tex]g''(x)=12x^2+2[/tex]

Sehingga diperoleh:

[tex]g'(x)=0\\4x^3+2x=0\\2x(2x^2+1)=0\\2x=0\vee 2x^2+1=0\\x=0[/tex]

Cek turunan kedua untuk mengetahui jenis stasionernya (x = 0)

[tex]g"(0)=12(0)^2+2=2 > 0[/tex]

Karena g"(0) > 0, maka jenis stasionernya adalah minimum.

Untuk titik minimumnya terletak pada (0, 3)

26. 5. Tentukan titik kritis, titik belok dan sketsa kurva f(x) = 2x³ + 3x² + 1​


Materi: Aplikasi Turunan Fungsi

Penjelasan dengan langkah-langkah:

+ Titik Kritis (Titik Balik)

Syarat memiliki nilai stasioner jika f'(x) = 0, maka :

[tex]6x^2+6x=0\\6x(x+1)=0\\x=0\vee\,x=-1[/tex]

Untuk menentukan titik balik minimum dan maksimumnya, kita dapat menggunakan uji turunan kedua.

f''(x) = 12x + 6

f"(0) = 6 (minimum)

f"(-1) = -6 (maksimum)

Jadi, titik - titik kritisnya adalah :

Minimum: (0,1)

Maksimum: (-1,2)

+ Titik Belok

Titik belok bisa diperoleh dengan turunan kedua dimana f"(x) = 0, maka :

12x + 6 = 0

12x = -6

x = -1/2

Berarti titik beloknya: (-1/2, 3/2)

30. 5. Diketahui : y = x³ - 6x² + 9x - 8 Tentukan : Titik-titik kritis​


Jawab:

Titik kritis adalah titik di mana turunan dari suatu fungsi bernilai nol atau tidak dapat ditentukan. Titik kritis dapat merupakan titik puncak atau titik lembah, atau juga titik infleksi.

Untuk mencari titik kritis dari fungsi y = x³ - 6x² + 9x - 8, pertama-tama kita perlu mencari turunan dari fungsi tersebut. Turunan fungsi yang diberikan adalah sebagai berikut:

y' = 3x² - 12x + 9

Setelah mencari turunan dari fungsi y, kita perlu mencari nilai x yang membuat turunan tersebut bernilai nol. Caranya adalah dengan menyelesaikan persamaan y' = 0.

Dengan menyelesaikan persamaan tersebut, kita akan mendapatkan titik kritis sebagai berikut:

x = 3/2

Jadi, titik kritis dari fungsi y = x³ - 6x² + 9x - 8 adalah (3/2, 0).
