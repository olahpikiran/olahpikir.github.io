10. Sebutkan Fungsi dari masing - masing isi item pada Menu Bar File (Info, New, Open, Save, Save As, Print, Share, Export, Close, Account, Option) ?


1.. Info.

Memunculkan tab informasi dokumen meliputi ukuran, judul, tag, kategori dokumen (Properties), tanggal pembuatan, dan pembaruan (Related Dates), penulis (Related People), dan mengunci memberikan pasword untuk melindungi privasi dokumen (Protect Workbook)

2.New.

Membuat dokumen baru, mencari dan menerapkan template atau tema online ke dalam project.

3.Open.

Membuka dokumen atau berkas yang sudah pernah dibuat, entah itu dalam komputer (This PC), atau One Drive.

4.Save.

Pada menu file microsoft excel, ikon atau perintah save memiliki fungsi untuk menyimpan dokumen ke dalam komputer maupun OneDrive. Shortcutnya: ctrl & S.

5.. Save As.

Perintah save as pada menu file microsoft excel ber-fungsi untuk menyimpan dokumen yang sudah ada dengan nama yang berbeda. Misalkan: Formulir 1 (Sudah ada), menjadi Formulir 2 (Nama lain yang sudah diisi). Sehingga terdapat 2 berkas yang berbeda. hal ini biasanya digunakan untuk sebagai backup antara file awal, dan yang sudah diedit, atau diperbarui.

6.Print.

Group Print pada menu file microsoft excel ber-fungsi untuk mengatur dokumen sebelum dicetak, meliputi jenis printer, halaman, margins, ukuran kertas, dan jumlah salinan. Akan muncul tab preview di samping kanan untuk mengetahui tampilan dokumen jika dicetak.

7.Share.

Share adalah ikon dalam menu File microsoft excel yang memilki fungsi membagikan dokumen ke orang lain melalui email, maupun OneDrive. Terdapat 2 opsi sharing anatar lain:

share pada menu file excel befungsi

Share with People.

Membagikan project yang dibuat ke orang lain yang terhubung dalam lingkup OneDrive.

Email.

Mengirimkan salinan dokumen melalui surat email, dalam bentuk lampiran.

Send a Link.

Menyispkan sebuah link yang berisi dokumen excel, untuk kebutuhan pengerjaan dokumen dengan banyak orang

8.Export.

Fungsi dari Export pada menu file microsoft excel adalah untuk mengubah format/type dokumen ke format lain seperti txt. Ada 2 sub bagian export antara lain:

export pada menu file

Create PDF/XPS Document.

Untuk membuat file menjadi tipe PDF/XPS.

Change File Type.

Mengubah lembar kerja ke dalam tipe dokumen lain seperti excel 97-2003, Spreadsheet, Txt, CVS dll dengan Save maupun Save As.

9.Close.

Fungsi utama Close yang merupakan bagian dari menu file adalah menutup dokumen yang sedang dikerjakan, tanpa mengakhir software microsoft excel.

10.Account.

Tab account yang dapat diakses melalui menu file excel, ber-fungsi untuk mengelola pengaturan akun, tema sistem, login office, dan informasi dari product office, seperti update terakhir, dan informasi software yang digunakan.

11.Options.

Fitur Options adalah sebuah fitur yang terdapat pada menu file excel, yang terdiri atas: General, Formulas, Data, Proofing, Save, Language, Ease of Acces, Advanced, Customize Ribbon, Quick Acces Toolbar, Add-ins, Trust Center, yang memilliki fungsi untuk mengatur toolbar seperti menyembunyikan, dan memunculkan Ribbon.

20. kan tentang sistem pembayaran internasional Open account dan Letter of Credit


Kelas : XI
Mapel : Ekonomi
Materi : Perdagangan Internasional
Sub materi : Pembayaran Internasional
 
•••••••••••••••••••••••••••••••••••••••••

Alat pembayaran internasional terbagi menjadi 2 yaitu pembayaran tunai dan nontunai sedangkan sistem pembayaran open account dan letter of credit termasuk kedalam alat pembayaran nontunai, berikut penjelasannya:
- Open account (pembayaran kemudian), yaitu pembayaran yang dilakukan setelah barang diterima importir dan tidak memerlukan surat perintah pembayaran atau dokumen tertentu. Jenis pembayarab ini diterapkan jika eksportir dan importir saling mengenal dengan baik, kondisi perekonomian stabil, dan lokasi produksi berdekatan dengan negara tujuan.
- Letter of credit (L/C) yaitu surat yang diterbitkan bank (issuing bank) atas permintaan importir ketika bank menyetujui akan membayarkan wesel kepada eksportir. L/C berfungsi sebagai alat pengganti kredit bank dan penjamin pembayaran bagi eksportir.
