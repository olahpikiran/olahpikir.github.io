4. jelaskan mekanisme kerja gerak biasa dan gerak refleks


Gerak biasa dilakukan dengan melalui :

(impuls) ⇒ reseptor  ⇒ syaraf sensorik ⇒  otak ⇒ syaraf motorik ⇒ efektor

Gerak refleks dilakukan dengan melalui

(impuls) ⇒ reseptor  ⇒ syaraf sensorik ⇒  tulang belakang ⇒ syaraf motorik ⇒ efektor

Keduanya memiliki kemiripan pada struktur namun memiliki perbedaan pada pusat respon yang berada pada otak dan tulang belakang. Gerak normal biasanya dilakukan melalui pertimbangan dan perintah daro otak, berbeda dengan gerak refleks yang dilakukan dalam jangka waktu yang singkat relatif terhadap impuls.

9. Jelaskan perbedaan mekanisme gerak refleks dan gerak sadar


"mengapa aktivitas melebar dan menyempitnya pupil karena rangsang cahaya di sebut sebagai gerak refleks otak? jelaskan melanismenya!"

Penjelasan Dengan Langkah-langkah :

Gerak Refleks

Berdasarkan letak neuron inteerkonektornya , gerak refleks dibagi 2, yaitu gerak refleks otak dan gerak refleks sumsum tulang belakang.

1) Gerak Refleks otak konektor (lintasan Refleks Cranialis) terjadi jika neuron penghubung yang dilalui impuls terdapat di otak.

Contohnya, Refleks mata (membuka dan menutupnya kelopak mata ketika kering) dan refleks pupil saat menyesuaikan rangsang cahaya yang masuk mata.

Mekanismenya sama dengan gerak sadar :

Rangsang  ⇒  neuron sensorik  ⇒  neuron pada saraf pusat (otak ) impuls tidak diolah ⇒ neuron motorik ⇒ efektor  ⇒  gerak Refleks

Reflek yang di integrasikan oleh otak.

2) Gerak Refleks sumsum Tulang Belakang Konektor (lintasan Refleks Spinalis), terjadi jika neuron penghubung yang dilalui impuls terdapat di sumsum tulang belakang.

Contohnya, gerak refleks ketika menyentuh benda panas.

Mekanisme gerak rekflek ini :

Rangsang ⇒ Neuron Sensorik ⇒ Neuron Penghubung pada saraf Pusat (sumsum tulang belakang) impuls tidak diolah  ⇒ neuron motorik ⇒ efektor ⇒ gerak Refleks.

Gerak Sadar

Atau juga biasa disebut dengan gerak biasa , Rangsang diterima oleh indera dan dihantarkan oleh saraf sensorik ke otak, Rangsang tersebut ditanggapi oleh otak dan diteruskan oleh sarafmotorik kealat tubuh dan terjadi gerak sadar.

mekanismenya sebagai berikut :

Rangsang  ⇒  Reseptor  ⇒  Neuron Sensorik  ⇒  Otak  ⇒  Neuron Motorik  ⇒  Efektor  ⇒  Gerak Biasa

15. jelaskan mekanisme gerak refleks dan gerak biasa


Gerak refleks dilakukan tanpa adanya impuls dari pusat syaraf di otak. Urutan gerak refleks adalah:

reseptor → neuron sensorik → sumsum tulang belakang → neuron motoris → efektor

Gerak biasa dilakukan akibat perintah yang dibawa impuls dari pusat syaraf di otak. Urutan gerak biasa:

Reseptor (organ indera) → neuron sensorik → interneuron di otak → neuron motoris → efektor

Pembahasan:  

Rangsangan yang diterima oleh tubuh kita akan dihantarkan oleh neuron. Rangsangan ini dapat berupa cahaya yang didengar oleh mata, bunyi yang didengar oleh telinga dan suhu atau tekstur permukaan yang diterima oleh kulit.

Rangsangan ditanggapi sebagai respons. Respons ini dapat berasal dari sistem saraf pusat di otak (untuk gerak biasa) atau sumsum tulang belakang (pada gerak refleks).

Gerak refleks misalnya adalah gerak menutup mata saat ada debu, sementara gerak biasa misalnya adalah egrak kaki melangkah atau tangan yang mengambil barang.

30. Jelaskan Mekanisme gerak sadar dan refleks


Kelas:  XII SMA

Mata Pelajaran: BIOLOGI

Kategori: SISTEM KOORDINASI

Kata Kunci: GERAK SADAR, GERAK REFLEK

 

 

Mekanisme gerak sadar dan reflex secara umum dapat dilihat pada skema berikut ini ya:    

Mekanisme gerak sadar:  

Rangsang ( impuls) -> reseptor ( alat indera) -> saraf sensorik ( aferen ) -> otak ( cranial) -> saraf motorik ( eferen ) -> efektor ( otot dan kelenjar ) -> tanggapan ( respon)  


Kemudian, untuk mekanisme gerak reflex :  

Rangsang ( impuls) -> reseptor ( alat indera) -> saraf sensorik ( aferen ) -> sum sum tulang belakang ( medulla spinalis ) -> saraf motorik ( eferen ) -> efektor ( otot dan kelenjar ) -> tanggapan ( respon)  

Nah, sebagai materi tambahannya, akan kita jabarkan jenis jenis gerak reflek ya…    

Gerak reflek dibagi menjadi 3 jenis lho..  

1.       Misalnya gerak reflek saat kita menginjak punting rokok, maka mekanisme gerak refleknya sebagai berikut ini:  

Rangsang ( impuls) -> reseptor ( alat indera) -> saraf sensorik ( aferen ) -> sum sum tulang belakang ( medulla spinalis ) -> saraf motorik ( eferen ) -> efektor ( otot dan kelenjar ) -> tanggapan ( respon)    

2.       Misalnya gerak reflek saat kita tersedak kemudian batuk atau muntah, maka mekanisme gerak refleknya sebagai berikut ini:  

Rangsang ( impuls) -> reseptor ( alat indera) -> saraf sensorik ( aferen ) -> sum sum lanjutan ( medulla oblongata ) -> saraf motorik ( eferen ) -> efektor ( otot dan kelenjar ) -> tanggapan ( respon)  

3.       Misalnya gerak reflek saat pupil kita membesar atau mengecil terkait dengan cahaya maka mekanisme gerak refleknya sebagai berikut ini:  

Rangsang ( impuls) -> reseptor ( alat indera) -> saraf sensorik ( aferen ) -> otak tengah ( mesencephalon ) -> saraf motorik ( eferen ) -> efektor ( otot dan kelenjar ) -> tanggapan ( respon)
