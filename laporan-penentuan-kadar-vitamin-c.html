6. Faktor apa saja yang mempengaruhi kadar vitamin c dalam suatu makanan


Kadar dari vitamin C, dipengaruhi oleh beberapa faktor, yaitu :
1.Keadaan buah : semakin layu/kusut atau tidak segarnya vitamin menyebabkan kadar vitamin C yang terkandung dalam buah tersebut berkurang.
2.Waktu pengekstraksian : semakin lama waktu mengekstrasi kandungan vitamin C akan semakinberkurang.
3.Masa penyimpanan : semakin lama suatu bahan disimpan, kadarnya akan semakin rendah.
4.Suhu : semakin tinggi suhu, kadarnya akan semakin rendah (Imma, 2009).

11. Diketahui dalam 100ml obat sirop mengandung 0,5mg vitamin C. Massa obat sirop tersebut 125gram. Tentukan kadar vitamin C dalam obat tersebut (bpj)


Diketahui dalam 100ml obat sirop mengandung 0,5mg vitamin C. Massa obat sirop tersebut 125gram.

 

Tentukan kadar vitamin C dalam obat tersebut (bpj)?

 

Pembahasan

Bagian per-juta (bpj), atau dalam bahasa inggris disebut ppm (part-per million) merupakan satuan konsentrasi larutan dimana 1 bpj yaitu konsentrasi 1 bagian partikel tertentu dalam setiap 1 juta bagian total partikel.  Umumnya satuan yang digunakan yaitu mg/Kg atau mg/L jika dilarutkan dalam air.


(bpj: 1 kg = 1000 gram = 1 juta mg atau nilainya sama dengan 0,000001).

 

Maka untuk menghitung kadar vitamin C dalam sirup menggunakan berat dibagi berat (w/w) yaitu:

Kadar vit C (bpj) = 0,5 mg/ 0,125kg = 4 bpj

 

Artinya dalam 1 juta bagian sirup mengandung 4 bagian vitamin C.

24. dalam 125 gram obat sirup terdapat 0,25 MG vitamin C Berapa kadar vitamin C dalam obat sirup tersebut


Kelas           : X
Mapel          : KIMIA
BAB            : STOIKIOMETRI

-----------------------------------------------------------------------------------------
Naskah Soal:
Dalam 125 gram obat sirup terdapat 0,25 mg vitamin C Berapa kadar vitamin C dalam obat sirup tersebut?

Diketahui:
Massa sirup : 125 gram
Massa Vitamin C = 0,25 mg = 0,00025 gram

Ditanya: % Kadasr Vitamin C dalam Sirup?

Jawab :
Persentase kadar zat dapat dihitung menggunakan rumus berikut:

                     %Kadar = (Massa Zat / Massa Total) x 100%

Jadi persentase kadar Vitamin C dalam sirup adalah:
  
                   % Vitamin C = (0,00025 gram/125 gram) x 100%
                   % Vitamin C = (0,000002) x 100%
                   % Vitamin C = 0,0002%

Jadi kadar vitamin C dalam sirup tersebut adalah 0,0002%
