18. Jika kalium nitrat dipanaskan Akan terbentuk kalium nitrit dan gas oksigen. Data tersebut jika ditulis dengan reaksi setara dan cara penyetaraannya bagaimana?


2 KNO3 ----> 2 KNO2 + O2

Dihitung jumlah atom2nya harus seimbang untuk penyetaraan

a KNO3 ---> b KNO2 + c O2

cara hitung koefisien dgn persamaan matematika

Lihat unsur/atom yg ada di sisi kiri & kanan, didapatkan sbb:

K: a = b
N: a = b
O: 3a = 2b + 2c

3a = 2a + 2c
a = 2c

jika c = 1 maka a = 2

diketahui a=b jadi b = 2

a = 2 ; b = 2 ; c = 1¤ menyetarakan reaksi

KNO3(aq) => aKNO2(aq) + bO2(g)

sebelum reaksi jumlah atom :
K = 1
N = 1
O = 3

sesudah reaksi jumlah atom :
K = a
N = a
O = 2a + 2b

sehinggan akan diperoleh hubungan
jumlah atom sebelum reaksi = jumlah atom sesudah reaksi

K sebelum reaksi = K sesudah reaksi

1 = a

N sebelum reaksi = N sesudah reaksi

1 = a

O sebwlum reaksi = O sesudah reaksi

3 = 2a + 2b

subatitusikan nilai a = 1

3 = 2(1) + 2b

3 = 2 + 2b

1 = 2b

b = 1/2

sehingga diperoleh

KNO3(aq) => KNO2(aq) + 1/2O2(g)
_______________________x2

2KNO2(aq) => 2KNO2(aq) + O2(g)

sudah setara
