8. Contoh kalimat silogisme negatif, dan silogisme error, lengkap dengan premis-premisnya.


Contoh kalimat silogisme negatif beserta premisnya yaitu:

Premis umum (PU): Semua penderita penyakit gula darah tidak boleh makan terlalu banyak gula.

Premis Khusus (PK): Luna menderita penyakit gula darah

Simpulan (S): Luna tidak boleh makan terlalu banyak gula.

Contoh kalimat silogisme error (salah) beserta premisnya yaitu:

PK: Lulu rajin menabung.

PK: Lulu menjadi kaya raya.

S: Orang yang rajin menabung akan kaya raya?

Dari contoh di atas tidak terdapat Premis Umum (PU).

Pembahasan:

Silogisme adalah sebuah proses berfikir yang bersebrangan dari satu atau lebih premis, yaitu, Pernyataan – pernyataan yang mengawalinya agar dapat diambil suatu kesimpulan berdasarkan prinsip – prinsip logis perlawanan ataupun pendasaran yang cukup.

Di dalam silogisme terdapat dua kategori premis dan satu kesimpulan. Kedua premis itu adalah premis umum (premis mayor) dan premis khusus (premis minor).

Premis Umum ( =PU ) menyatakan jika seluruh anggota golongan tertentu (= semua A) mempunyai sifat ataupun hal tertentu ( =B ). Premis Khusus ( =PK ) menyatakan jika sesuatu ataupun seseorang ( =C ) merupakan suatu anggota golongan tertentu itu ( =A ).

Sedangkan simpulan ( =S ) menyatakan jika sesuatu atau seseorang itu ( =C ) mempunyai sifat ataupun hal tersebut pada B ( =B ).

Jenis Jenis Silogisme.

Silogisme Negatif

Silogisme Negatif bisa dilihat dengan adanya kata “bukan atau tidak” pada premis dan begitu juga simpulan. Jika satu premis pada silogisme bersifat negatif, maka simpulannya bersifat negatif juga.

Contoh :

PU : Semua yang masuk sekolah tidak boleh telat.

PK : Rino masuk sekolah.

S : Rino tidak boleh telat.

Silogisme Error (salah)

Dalam sebuah argumentasi kita harus berhati-hati menggunakan penalaran silogisme. Jika tidak berhati-hati maka akan timbul masalah ketika mengambil simpulan. Jika merumuskan premis harus dengan cermat sebab kesalahan sering terjadi pada penyusunan premis. Untuk menghindari kesalahan perlu diperhatikan peringatan dibawah ini :

Jika terdapat dua premis khusus, maka tidak bisa ditarik simpulan yang dapat dipertanggung jawabkan kebenarannya

PK: Anang diterima sebagai mahasiswa UI.

PK: Anang remaja yang cerdas.

S: Remaja yang cerdas diterima sebagai mahasiswa UI?

Dari contoh tersebut di atas tidak terdapat PU.

Pendahuluan

Jenis Jenis LogikaMatematika :

1. Pernyataan

kalimat yang tertutup yang memiliki nilai benar atau salah saja tapi tidak sekaligus benar dan salah. Contoh :

“hari ini hari senin, maka hari ini tidak libur”

“Musuh saya sudah kalah, saya senang”

2. Kalimat Terbuka

kalimat yang belum pasti nilai kebenarannya. Contoh :

“jika saya membantu ibu dua kali dan membantu paman sekali, saya dapat permen”

“harga empat ekor kucing dan seekor harimau adalah tiga ratus ribu”

3. Ingkaran (Negasi)

Ingkaran (~) adalah suatu operasi yang membalikan suatu pernyataan. Contoh :

A = “hari ini saya libur”

~A = “hari ini saya tidak libur”

B = “Uang saya masih banyak”

~B = “Uang saya tinggal sedikit”

Pernyataan Majemuk

1. Konjungsi (^)

Bernilai benar jika pernyataan lainnya bernilai benar semua. Contoh

P = B

Q = S

P ^ Q = S karena P ^ Q hanya bernilai benar jika P dan Q bernilai benar.

R = B

S = B

R ^ S = B karena nilai R dan S sama sama benar

2. Disjungsi (V)

bernilai benar jika salah satu dari pernyataan bernilai benar. Contoh :

P = S

Q = S

P V Q = S karena tidak ada yang bernilai benar

R = S

S = B

R V S = B karena salah satu dari pernyataannya bernilai benar

3. Implikasi (=>)

bernilai salah jika pernyataan pertama bernilai benar, pernyataan kedua bernilai salah, dan seterusnya. Contoh :

P = B

Q = S

P => Q = S karena permyataan pertama P bernilai benar dan pernyataan kedua Q bernilai salah

A = S

C = B

A => C = B karena pernyataan pertama bernilai salah

4. Biimplikasi (<=>)

Bernilai benar jika kedua pernyataan memiliki kebenaran yang sama. Contoh :

P = B

Q = S

P <=> Q = S karena dua duanya berbeda yaitu P benar Q salah

B = S

C = S

B <=> C = B karena B dan C sama sama salah

G = B

H = B

G <=> H = B karena G dan H sama sama benar

Pernyataan majemuk yang Ekuivalen

1. Komutatif

p ^ q = q ^ p

p V q = q V p

2. Asosiatif

p ^ (q ^ r) = (p ^ q) ^ r

p V (q V r) = (p V q) V r

3. Distributif

p ^ (q V r) = (p ^ q) V (p ^ r)

p V (q ^ r) = (p V q) ^ (p V r)

4. Eksportasi

[(p ^ q) => r] = [p => (q => r)]

5. Hukum De Morgan

~(p V q) = ~p ^ ~q

~(p ^ q) = ~p V ~q

6. Konjungsi

p ^ q = ~(~p V ~q) = ~(p => ~q)

7. Disjungsi

p V q = ~(~p ^ ~q) = ~p => q

8. Implikasi

p => q = ~(p ^ ~q) = ~p V q

9. Negasi dari Implikasi

~(p => q) = p V ~q

PENARIKAN KESIMPULAN DALAM LOGIKA MATEMATIKA

Dalam logika matematika ada tiga cara penarikan kesimpulan, yaitu:

Modus ponenModus tolensSilogisme

Modus ponen memiliki bentuk penarikan kesimpulan:

p ⇒ q

p        

q

Contoh:

Jika ibu duduk melamun maka ibu teringat akan almarhum ayah.

Ibu duduk melamun.

Kesimpulan : Ibu teringat akan almarhum ayah.

Modus tolens memiliki bentuk penarikan kesimpulan:

p ⇒ q

~q        

~p

Contoh:

Jika adik mendapat hadiah maka adik akan berlari kegirangan.

Adik tidak berlari kegirangan.

Kesimpulan : Adik tidak mendapat hadiah

Silogisme memiliki bentuk penarikan kesimpulan:

p ⇒ q

q ⇒ r  

p ⇒ r

Contoh:

Jika nilai matematika saya jelek maka saya harus les matematika.

Jika saya les matematika maka saya tidak dapat bermain game.

Kesimpulan:

Jika nilai matematika saya jelek maka saya tidak dapat bermain game.

        Premis umum (PU) : Berupa pertanyaan atau kumpulan kelompok tertentu dimana memiliki sifat, watak atau ciri tertentu.

        Premis khusus (PK) : Pernyataan tentang suatu anggota dari kelompok tersebut.

        Kesimpulan : Kesimpulan bahwa satu anggota dari kelompok tersebut memiliki sifat atau ciri tertentu berdasarkan pernyataan dari premis umum.

Rumus Silogisme

        PU terdiri dari A = B

        PK terdiri dari C = A

        K terdiri dari C = B

Contoh Silogisme

        PU : Mamamlia adalah hewan menyusui

       PK  : Sapi adalah mamalia

         K  : Maka sapi adalah hewan menyusui

Paragraf Silogisme

        Semua mamalia di dunia ini adalah hewan menyusui. Sapi termasuk dari hewan mamalia karena memiliki kelenjar mamae. Maka dapat disimpulkan bahwa sapi termasuk dari hewan mamalia.

Penjelasan dalam soal.

Didalam soal diketahui bahwa B identik C (B = C), A termasuk himpunan B (A=B), dan A termasuk juga himpunan C (A =C). Maka dapat ditelis rumus silogisme seperti ini:

PU = terdiri dari B = C

PK = terdiri dari A = B

K  = terdiri dari A = C

Untuk membuat contoh silogisme kita harus membuat premis utama dalam keterangannya. Dengan kata yang memiliki hubungan dengan premis, contoh:

A = Sapi  ;   B = Mamalia   ;   C = Menyusui

Sehingga kita bisa membuat contoh silogisme seperti:

PU   = Mamalia adalah hewan menyusui (B = C)

PK   = Sapi adalah mamalia (A = B)

K     = Sapi adalah hewan menyusui (A = C)
