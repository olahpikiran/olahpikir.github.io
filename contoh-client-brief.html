1. menganalisis client brief, berserta contoh dan penjelasan​


Jawaban:

Client brief adalah dokumen atau formulir yang digunakan oleh sebuah perusahaan atau agensi untuk mengumpulkan informasi tentang proyek atau layanan yang diminta oleh klien. Dokumen ini berisi detail mengenai tujuan, ruang lingkup, target pasar, pesan utama, dan preferensi visual klien.

Berikut ini adalah contoh client brief beserta analisisnya:

Contoh Client Brief

Nama Klien: PT ABCD

Nama Proyek: Kampanye Pemasaran Baru

Tujuan: Meningkatkan penjualan produk di pasar regional

Target Pasar: Orang dewasa berusia 25-45 tahun, pendapatan menengah ke atas

Pesan Utama: Produk PT ABCD berkualitas dan aman digunakan

Preferensi Visual: Desain yang modern dan elegan, warna dominan hitam dan putih, tampilan minimalis

Analisis

Nama klien: PT ABCD. Informasi ini memberi tahu kami tentang identitas klien.Nama Proyek: Kampanye Pemasaran Baru. Informasi ini memberi tahu kami tentang jenis proyek yang diminta oleh klien.Tujuan: Meningkatkan penjualan produk di pasar regional. Informasi ini memberi tahu kami tentang tujuan akhir dari proyek ini.Target Pasar: Orang dewasa berusia 25-45 tahun, pendapatan menengah ke atas. Informasi ini memberi tahu kami tentang siapa yang harus dijadikan target pasar dalam kampanye pemasaran ini.Pesan Utama: Produk PT ABCD berkualitas dan aman digunakan. Informasi ini memberi tahu kami tentang pesan utama yang ingin disampaikan dalam kampanye pemasaran.Preferensi Visual: Desain yang modern dan elegan, warna dominan hitam dan putih, tampilan minimalis. Informasi ini memberi tahu kami tentang preferensi visual klien untuk desain kampanye pemasaran baru.

Dengan memahami client brief, kita bisa menghasilkan proyek yang lebih efektif dan efisien. Informasi yang diberikan dalam client brief dapat membantu perusahaan atau agensi untuk memahami kebutuhan klien dengan lebih baik dan menghasilkan hasil akhir yang memuaskan.

11. bagaimana pendapatmu tentang contoh software ftp client dan software ftp server​


Jawaban:

Aplikasi FTP Client:

1. FileZilla (Windows)

FileZilla menjadi salah satu software FTP Client yang cukup populer di kalangan pengguna Windows. Walaupun software ini juga dapat diaplikasikan pada Linux maupun Mac, sistem kerjanya memang lebih terintegrasi bila digunakan pada Windows.  

Keunggulan:

Sistem penggunaannya yang sangat mudah, yakni hanya dengan memasukkan alamat web, nama pengguna, dan kata kunci FTP, FTPS dan SFTP.  

Dapat mengupload file dalam format RAR, ZIP, dan EXE dengan kapasitas maksimal 4 GB.  

Pengiriman data yang fleksibel ke berbagai cloud provider seperti Google Drive, DropBox, dan OneDrive.

Kekurangan:

Bila kapasitasnya lebih dari 4 GB, kemungkinan besar proses mengupload akan memakan waktu lebih lama.

2. WinSCP (Windows)

Aplikasi FTP Client terbaik lainnya selain FileZilla adalah WinSCP. FTP Client yang dirancang khusus untuk Windows ini juga menjadi opsi yang sangat kuat berkat tampilan interface yang sederhana sehingga pemakaiannya menjadi sangat mudah.  

Keunggulan:

WinSCP mampu mengoperasikan beberapa protokol seperti SFTP, SCP, FTP, dan WebDev.

Dibekali dengan dukungan SSH password, keyboard-interactive, kunci publik, Kerberos (GSS) otentikasi, serta terintegrasi dengan Pageant yang didukung penuh oleh otentikasi kunci publik dengan SSH.  

Kekurangan:

Hanya dapat digunakan oleh pengguna sistem operasi Windows saja.

3. FireFTP (Windows, Mac OS X dan Linux)

Berbeda dengan FTP Software pada umumnya yang memerlukan beberapa tahap instalasi, penginstalan FireFTP hanya sebatas pengunduhan add-on pada browser Mozilla Firefox Anda. Namun, pastikan bila Mozilla Firefox Anda telah terupdate sebelum menginstal versi FireFTP terbaru.  

Keunggulan:

Sama dengan FileZilla, FireFTP juga bisa diaplikasikan bagi Anda para pengguna Mac dan Linux.  

Dirancang dengan interface yang user-friendly alias mudah digunakan.  

FireFTP selalu memberikan keamanan penuh dengan metode enkripsi terbaru termasuk SSL, SFTP, dan TLS.  

Menawarkan fitur kompresi file dengan MODE Z sehingga prosesnya menjadi lebih cepat dan singkat.  

Kekurangan:

Beberapa pengguna mengeluhkan bila proses kompresi file mereka kerap kali error dan lambat bila file tersebut melebihi kapasitas 4 GB.

Aplikasi FTP Server:

1. FileZilla Server

Aplikasi FileZilla Server ini gratis dan merupakan FTP Server yang bersifat open source. Source code FileZilla Server berada di SourceForge.net.  Beberapa fitur yang ada di dalam aplikasi ini adalah:  

Kompresi dengan DEFLATE (MODE Z) Enkripsi dengan SSL / TLS (untuk FTPS) Izin per pengguna pada sistem file Alat konfigurasi GUI Speed limits

2. Microsoft IIS

Microsoft IIS atau biasa disebut sebagai IIS hadir pada bulan Januari 2010 lalu yang ditambahkan oleh PierreG dan pada bulan Mei 2019 lalu aplikasi ini mengalami pembaharuan.

Internet Information Service atau disingkat IIS yang diperuntukkan Windows server merupakan sesustu yang aman, fleksibel, dan mudah untuk mengelola web server untuk menghosting apapun di web. Mulai dari streaming media hingga hosting aplikasi web, Microsoft IIS siap untuk menangani tugasnya.

3. Syncplify.me Server!

Server Syncplify.me! merupakan aplikasi FTP, FTPS, dan SFTP gratis yang dapat digunakan di sistem operasi windows. Aplikasi ini dibuat oleh Syncplify, Inc. Anda tidak perlu khawatir dengan aplikasi ini karena aplikasi ini sudah terjamin aman, powerful, dan sangat mudah untuk dikonfigurasi.

Beberapa fitur utama yang terdapat di dalam aplikasi ini adalah dukungan untuk FTP biasa, FTPS eksplisit (TLS), FTPS implisit (SSL / TLS), dan protokol transfer file SFTP, basic secure shell (SSH-2) serta mendukung file dengan ukuran yang besar (> 4 GB). APlikasi ini benar-benar gratis, termasuk dengan peng-update an dan peningkatan seumur hidup, khusus untuk penggunaan pribadi.


12. pengertian aplikasi client server serta contoh dan penjelasannya


Client-Server adalah pembagian kerja antara server dan client yang mengakses server dalam suatu jaringan.

Contohnya:

1. File Servers

File server vendors mengklaim bahwa mereka pertama menemukan istilah client-server.

Untuk sharing file melalui jaringan

2. Database Servers

Client mengirimkan SQL requests sebagai pesan pada database server,selanjutnya hasil perintah SQL dikembalikan.

Server menggunakan kekuatan proses yang diinginkan untuk menemukan data yang diminta dan kemudian semua record dikembalikan pada client.

3. Transaction Servers (Transaksi Server)

Client meminta remote procedures yang terletak pada server dengan sebuah SQL database engine.

Remote procedures ini mengeksekusi sebuah grup dari SQL statement

Hanya satu permintaan / jawaban yang dibutuhkan untuk melakukan transaksi

4. Groupsware Servers

Dikenal sebagai Computer-supported cooperative working

Manajemen semi-struktur informasi seperti teks, image, , bulletin boards dan aliaran kerja

Data diatur sebagai dokumen

5. Object Application Servers

Aplikasi client/server ditulis sebagai satu set objek komunikasi

Client objects berkomunikasi dengan server objects melalui Object Request Broker (ORB)

Client meminta sebuah method pada remote object

6. Web Application Servers (Aplikasi Web Servers)

World Wide Web adalah aplikasi client server yang pertama yang digunakan untuk web.

Client dan servers berkomunikasi menggunakan RPC seperti protokol yang disebut HTTP.

22. Berikut ini yang bukan termasuk contoh aplikasi FTP Client adalah….​


Jawaban:

File Transfer Protocol atau FTP adalah salah satu jenis protokol yang bertugas sebagai perantara dalam proses pertukaran data antar komputer dengan server melalui bantuan jaringan internet. Keberadaan FTP akan memfasilitasi Anda untuk melakukan berbagai aktivitas seperti mengunduh, mengunggah, menghapus, dan menambahkan file dengan mudah. Selain itu, FTP juga memfasilitasi Anda untuk melakukan remote server, alias pengelolaan website tanpa perlu masuk ke dalam akun hosting atau Virtual Private Server (VPS).

Terdapat dua jenis FTP yang perlu Anda ketahui yaitu FTP Server dan FTP Client. FTP Server merupakan server yang berfungsi untuk menjalankan software (aplikasi) saat menerima berbagai permintaan tukar menukar file yang diminta oleh FTP Client. Sedangkan, FTP Client itu sendiri adalah perangkat komputer yang Anda gunakan untuk melakukan segala aktivitas.

Untuk informasi lebih lanjut mengenai aplikasi FTP, Anda bisa membaca artikel lengkap kami di Apa itu FTP dan Bagaimana Cara Menggunakan FTP?

Fungsi FTP

FTP sangat mendukung untuk melakukan pertukaran file dalam ukuran sebesar apa pun. Proses dengan FTP pun jauh lebih cepat bila dibandingkan dengan proses download dan upload yang dilakukan di internet. Selain itu, resiko kehilangan file saat gagal di tengah proses mengunduh atau mengunggah pun tidak akan terjadi karena file Anda akan selalu ter-back up secara otomatis.

Untuk selengkapnya, berikut ini kami jabarkan beberapa fungsi yang bisa Anda dapatkan dari penggunaan FTP:

1. Keamanan

FTP telah dilengkapi dengan sistem keamanan enkripsi pada jaringan yang menjanjikan data sensitif Anda akan terproteksi secara penuh. Prinsip penggunaannya pun menggunakan metode autentikasi yang mengharuskan Anda memasukkan username dan password untuk bisa mengakses berbagai data yang terdapat pada FTP Server.

Sehingga, bila setiap kali Anda hendak mengunduh, menggugah dan melakukan aktivitas lainnya, username dan password tersebutlah yang akan menjadi pintu akses Anda untuk membuka direktori file yang terdapat didalamnya.

2. Sistem Kontrol

Anda bisa melakukan sistem kontrol secara bebas dalam pengaksesan aplikasi FTP atas data Anda.  Misalnya seperti di sebuah perusahaan, tidak semua karyawan memiliki akses yang sama untuk menggunakan FTP. Anda bisa mengatur siapa saja yang memiliki akses untuk menggunakan FTP, maupun membatasi pengaksesan pada data-data tertentu.

3. Ukuran Data yang Besar

Mungkin banyak di antara Anda yang mengalami kesulitan ketika mengirimkan data berukuran besar melalui email. Sebab hampir setiap layanan email memang hanya menyanggupi pengiriman data maksimal 25 MB saja. Maka dari itu, FTP bisa menjadi solusi terbaik untuk Anda yang ingin melakukan pengiriman file berukuran besarsekaligus dalam waktu cepat.

4. Pengiriman Data yang Terjadwal

Secara tidak langsung, penggunaan aplikasi FTP Client dapat meningkatkan efisiensi waktu Anda dalam bekerja. Sebab, Anda dapat menjadwalkan setiap data yang ingin dikirimkan tanpa perlu menyita waktu produktif Anda.

5. Otomatis Backup

Setiap aplikasi FTP Client telah dibekali dengan fitur backup otomatis. Jadi apabila sewaktu-waktu terjadi masalah seperti koneksi internet yang hilang, mati listrik, atau kerusakan pada PC, Anda tak perlu khawatir untuk kehilangan data. Sebab semua data penting Anda

The sprint is a five-day process for answering critical business questions through design, prototyping, and testing ideas with customers. Developed at GV, it’s a “greatest hits” of business strategy, innovation, behavior science, design thinking, and more — packaged into a battle-tested process that any team can use.

Instead of waiting to launch a minimal product to understand if an idea is any good, you’ll get clear data from a realistic prototype.
