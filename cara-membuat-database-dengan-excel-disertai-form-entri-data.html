Cara untuk membuat grafik pada microsoft excel adalah dengan melakukan seleksi pada sebuah tabel yang berisi data yang akan kita buat menjadi grafik. Setelah data terblok/ terseleksi dengan benar langkah selanjutnya adalah memilih menu Insert, kemudian pilih charts. Pilih chart atau grafik sesuai dengan keinginan kita.

Pembahasan

Pembuatan grafik merupakan salah satu fitur dari microsoft excel. Dengan memakai grafik, penyajian data akan semakin mudah dan lebih nyaman untuk dibaca. Pada microsoft excel terdapat beberapa macam jenis grafik yang bisa kita gunakan. Jenis-jenis grafik pada microsoft excel adalah sebagai berikut

Column ChartPie ChartLine ChartBar ChartArea Chart Scatter Chart Stock ChartsDoughnut charts Surface Chart Radar Chart (Spider Chart)Buble Chart

Setelah kita mengetahui jenis-jenis grafik yang ada pada microsoft excel, maka kita bisa memilihnya untuk kebutuhan penyajian data kita. Penggunaan grafik yang tepat akan mempermudah kita dalam menyajikan data.

3.export.php
<?PHP
  include "config.php";
  function cleanData(&amp;$str){
    $str = preg_replace("/\t/", "\\t", $str);
    $str = preg_replace("/\r?\n/", "\\n", $str);
    if(strstr($str, '"')) $str = '"' . str_replace('"', '""', $str) . '"';
  }

  // Nama Hasil Export
  $filename = "hasil_export_" . date('dmY') . ".xls";

  header("Content-Disposition: attachment; filename=\"$filename\"");
  header("Content-Type: application/vnd.ms-excel");

  $flag = false;
  $result = mysql_query("SELECT * FROM data ORDER BY id") or die('Query failed!');
  while(false !== ($row = mysql_fetch_assoc($result))) {
   if(!$flag) {
      echo implode("\t", array_keys($row)) . "\r\n";
      $flag = true;
    }
    array_walk($row, 'cleanData');
    echo implode("\t", array_values($row)) . "\r\n";
  }
  exit;
?>
