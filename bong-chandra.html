16. Uang Andri 2/3 uang Chandra dan uang Chandra 4/5 uang Burhan. Jika jumlah uang Burhan dan Chandra Rp540.000,00 maka selisih uang Andri dan Chandra adalah …​


A = Andri

B = Burhan

C = Chandra

A = 2/3 C

C = 4/5 B --> B = 5/4 C

B + C = 540.000

5/4 C + C = 540.000

9/4 C = 540.000

9 C = 2.160.000

C = 240.000 (uang Chandra)

A = 2/3 C = 2/3 x 240.000 = 160.000 (uang Andri)

Selisih uang Andri dan Chandra = 240.000 - 160.000 = 80.000
