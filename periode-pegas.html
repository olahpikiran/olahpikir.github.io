17. Pegas A dengan konstanta 300 N/m dan pegas B dengan konstanta 600 N/m. Pegas A diberi beban dengan massa 0,5 kali massa pada pegas B, jika kedua pegas digetarkan, maka perbandingan periode pegas A dengan periode pegas B adalah​


>>Gerak Harmonik Sederhana: Pegas<<

Mencari Periode

[tex]\boxed{\boxed{T=2\pi \sqrt{\frac{m}{k} } }}\\\boxed{\boxed{T=\frac{1}{f}}}[/tex]

Mencari Frekuensi

[tex]\boxed{\boxed{f=\frac{1}{2\pi }\sqrt{\frac{k}{m} } }}\\\boxed{\boxed{f=\frac{1}{T} }}[/tex]

dimana:

T = Periode (s)

f = Frekuensi (Hz)

k = konstanta pegas (N/m)

m = massa (kg)

Diketahui:

ka = 300 N/m

kb = 600 N/m

ma = 0,5mb

Ta : Tb ?

Jawab:

[tex]\frac{T_a}{T_b} =\frac{2\pi \sqrt{\frac{m_a}{k_a} } }{2\pi \sqrt{\frac{m_b}{k_b}} } \\\frac{T_a}{T_b} =\frac{\sqrt{\frac{m_a}{k_a} } }{\sqrt{\frac{m_b}{k_b}} } \\\frac{T_a^2}{T_b^2} =\frac{\frac{m_a}{k_a} }{\frac{m_b}{k_b} } \\\frac{T_a^2}{T_b^2} =\frac{\frac{0,5m_b}{k_a} }{\frac{m_b}{k_b} } \\\frac{T_a^2}{T_b^2} =\frac{\frac{0,5}{300} }{\frac{1}{600} } \\\frac{T_a^2}{T_b^2} =\frac{\frac{1}{600} }{\frac{1}{600} } \\\frac{T_a^2}{T_b^2} =\frac{1}{1 } \\\frac{T_a}{T_b} =\frac{1}{1 } \\[/tex]

[tex]T_a:T_b\\\boxed{1:1}[/tex]

18. dalam sistem getaran pegas bermassa, jika massa beban pegas ditambah maka nilai frekuensinya akan...... dan nilai periodenya akan......., jika pegas akan di ganti dengan pegas lain yang konstanta pegas/nilai kekakuannya lebih besar maka nilai frekuensinya akan...... dan nilai periodenya akan.......


ini dianalisa menggunakan hukum kekekalan energi

Kita kan tau pada sistem pegas bermassa :
1) Saat massa pegas hendak berbalik arah, dia akan berhenti terlebih dahulu. Sehingga, Energi kinetik pada saat ini adalah 0, dan energi potensial pegasnya adalah maksimal
2) Saat massa pegas berada pada "titik tengah"-nya, Energi kinetiknya akan mencapai maksimum dan energi potensial pegasnya bernilai 0
3) Dan saat posisinya berada di titik kecuali titik yang telah disebutkan tadi, sistem pegas tersebut akan memiliki nilai pada Energi kinetik dan potensial pegas, namun tidak maksimum

Nah, untuk menjawab pertanyaan, kita dapat menggunakan hukum kekekalan energi sistemnya pada titik "pegas hendak berbalik arah" dan saat pegas memiliki nilai energi kinetik dan potensial pegas :

Epegas + Ek = Epegas max + Ek
1/2.k. x^2 + 1/2 m v^2 = 1/2 k (x max)^2 +0
(Lalu dengan perpindahan ruas, didapatkan :

v^2 = [k((x max)^2 - x^2)]/m

Berdasarkan persamaan itu, nilai v berbanding terbalik dengan nilai m (semakin besar nilai m, senakin kecil nilai v)

* Lalu menggunakan sifat gelombang
v = lamda. f
T = 1/f

Dapat DISIMPULKAN bahwa jika nilai m bertambah, nilai v akan semakin kecil dan nilai f pula akan semakin kecil, tetapi nilai periode semakin besar


DAN jika NILAI K LEBIH BESAR, berdasarkan persamaan tadi, hal sebaliknya yg akan terjadi, nilai v akan semakin besar dan nilai f....dst

25. Jika periode suatu pegas 2 s, maka saat beban pada pegas dijadikan 4 kali semula, maka periodenya menjadi ...


Bab : Getaran

Periode getaran pegas, T dihitung dg rumus:
T = 2π √(m/k)
dengan
T = periode
m = massa beban
k = konstanta pegas

Dari rumus diatas didapat hubungan antara periode dengan massa beban dan konstanta pegas.
Dimana periode getaran pegas berbanding lurus dengan √m dan berbanding terbalik √k

Pembahasan soal:
Diketahui:
m1 = m
T1 = 2 s
m2 = 4m

Ditanya:
T2 = ...?

Dijawab:
T2/T1 = √(m2/m1)
T2/2 = √(4m/m)
T2 = √4 x 2
T2 = 2 x 2
T2 = 4 s

Ditanya :
T2 = ?

Penyelesaian :
Pegas bergetar secara harmonik sehingga menimbulkan getaran dengan periode tertentu.

Gunakan rumus periode pegas :
T = 2π √m/k

T = 2π √m/k
T² = (2π)² (√m/k)²
T² = 4π² m/k
T²/m = 4π²/k

Pada peristiwa ini, konstanta pegas tetap atau tidak berubah. Sedangkan periode dan massa beban berubah

T²/m = Konstan
T1²/m1 = T2²/m2
(5)²/m = T2²/2m
25 = T2²/2
T2² = 50
T2 = √50
T2 = √(25.2)
T2 = 5√2 sekon
T2 = 7,07 sekon

-> Jadu, periode pegas menjadi 5√2 sekon atau 7,07 sekon
